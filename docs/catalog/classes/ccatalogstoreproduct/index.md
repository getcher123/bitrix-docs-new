# CCatalogStoreProduct

**CCatalogStoreProduct** - класс для работы со остатками товара на складах.


#### Методы класса


| Метод | Описание | С версии |
| --- | --- | --- |
| Add | Метод добавляет остаток товара, в соответствии с данными из массива arFields. | 12.0.0 |
| Delete | Метод удаляет запись из таблицы остатков товара с кодом ID. | 12.0.0 |
| GetList | Метод возвращает результат выборки записей из таблицы остатков товара в соответствии со своими параметрами. | 12.0.0 |
| Update | Метод изменяет параметры записи с кодом ID, в соответствии с данными из массива arFields. | 12.0.0 |
| UpdateFromForm | Метод изменяет параметры записи. | 12.0.0 |

# CCatalogProduct


#### Методы класса


| Метод | Описание | С версии |
| --- | --- | --- |
| [Add](add.md) | Метод добавляет (или обновляет) параметры товара к элементу каталога. | 3.3.5 (устарел с 17.6.0) |
| clearUsedCurrency | Метод указывает, что для расчета наименьшей цены в методе CCatalogProduct::GetOptimalPrice необходимо использовать базовую валюту. | 15.0.2 |
| CountPriceWithDiscount | Метод применяет к цене price в валюте currency наибольшую скидку из массива arDiscounts. | 4.0.4 |
| [Delete](ccatalogproduct__delete.ed301fc8.md) | Метод удаляет из элемента каталога свойства, относящиеся к товару. | 3.3.5 (устарел с 17.6.0) |
| GetList | Метод возвращает результат выборки записей товаров в соответствии со своими параметрами. | 3.3.5 (устарел с 17.6.0) |
| GetByIDEx | Метод по коду товара ID возвращает все параметры этого товара, включая относящиеся к товару как к элементу информационного блока. | 3.3.5 |
| GetByID | Метод по коду товара ID возвращает массив параметров товара (которые относятся к товару как к таковому). | 3.3.5 |
| GetNearestQuantityPrice | Метод по коду товара ID возвращает количество товара, доступное для покупки. | 4.0.4 |
| GetOptimalPrice | Метод возвращает параметры наименьшей цены товара productID в количестве quantity для посетителя, входящего в группы пользователей arUserGroups. | 4.0.4 |
| getUsedCurrency | Метод возвращает текущую валюту, в которой считается наименьшая цена в методе CCatalogProduct::GetOptimalPrice. | 15.0.2 |
| IsExistProduct | Метод проверяет наличие информации о товаре с кодом intID. | 14.0.1 |
| QuantityTracer | Уменьшает количество на складе товара с кодом PRODUCT_ID на величину DELTA_QUANTITY, если указанное количество есть на складе и у товара установлен флаг "уменьшать количество товара при заказе". | 3.3.5 |
| [Update](ccatalogproduct__update.bc9a623b.md) | Метод обновляет параметры товара, относящиеся к товару как к таковому. | 3.3.5 (устарел с 17.6.0) |

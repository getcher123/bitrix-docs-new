# Delete


```
bool
CCatalogProduct::Delete(
	int ID
);
```

Метод удаляет из элемента каталога свойства, относящиеся к товару. Нестатический метод.


#### Параметры вызова


| Параметр | Описание |
| --- | --- |
| ID | Код товара (элемента каталога). |


#### Возвращаемые значения

Возвращает *true* в случае успешного удаления и *false* в противном случае.

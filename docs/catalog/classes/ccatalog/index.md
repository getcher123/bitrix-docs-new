# CCatalog


#### Методы класса


| Метод | Описание | С версии |
| --- | --- | --- |
| CheckFields | Метод проверяет параметры, переданные в методы Add и Update. | 4.0.4 |
| GetList | Метод возвращает результат выборки записей из каталога в соответствии со своими параметрами. | 3.2.1 |
| Update | Метод изменяет параметры записи с кодом ID в таблице привязывания информационного блока к модулю Торгового Каталога. | 3.2.1 |
| Add | Метод служит для добавления новой записи в таблицу привязывания информационного блока к модулю Торгового Каталога. | 3.2.1 |
| Delete | Метод удаляет привязку информационного блока с кодом ID к торговому каталогу. При этом удаляются так же параметры товаров и предложения, относящиеся к этому каталогу. Описания товаров, относящиеся к элементу информационного блока, остаются неизменными. | 3.2.1 |
| GetByID | Возвращает массив параметров каталога, включая некоторые параметры, относящиеся к информационному блоку. | 3.2.1 |

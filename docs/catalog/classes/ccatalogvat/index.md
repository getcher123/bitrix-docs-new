# CCatalogVat

**CCatalogVat** - класс для работы со ставками НДС.


#### Методы класса


| Метод | Описание | С версии | До версии |
| --- | --- | --- | --- |
| Add | Метод добавляет новую ставку НДС в соответствии с данными из массива *arFields*. | 12.5.6 |  |
| CheckFields | Метод служит для проверки параметров, переданных в методы CCatalogVat::Add и CCatalogVat::Update. | 7.0.0 |  |
| Delete | Метод удаляет ставку НДС с кодом *ID*. | 7.0.0 |  |
| GetByID | Метод возвращает ставку НДС по ее коду *ID*. | 7.0.0 |  |
| GetList | Метод возвращает результат выборки записей из таблицы ставок НДС в соответствии со своими параметрами. | 7.0.0 | 12.5.6 |
| GetListEx | Метод возвращает результат выборки записей из таблицы ставок НДС в соответствии со своими параметрами. | 12.5.6 |  |
| Set | Метод добавляет новую ставку НДС или обновляет существующую в зависимости от передаваемых данных в массиве *arFields*. | 7.0.0 | 12.5.6 |
| Update | Метод изменяет параметры ставки НДС с кодом *ID* в соответствии с данными из массива *arFields*. | 12.5.6 |  |

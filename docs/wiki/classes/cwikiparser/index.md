# CWikiParser

**CWikiParser** - Класс обрабатывающий вики-разметку в странице.


#### Методы класса


| Метод | Описание | С версии |
| --- | --- | --- |
| Parse | Обрабатывает содержимое Wiki-страницы перед отображение. |  |
| parseBeforeSave | Обрабатывает содержимое Wiki-страницы перед сохранением. |  |
| parseForSearch | Обрабатывает содержимое Wiki-страницы для модуля «Поиск». |  |

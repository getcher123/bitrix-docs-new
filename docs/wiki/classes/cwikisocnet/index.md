# CWikiSocnet

**CWikiSocnet** - Класс интеграции с модулем «Социальная сеть».


#### Методы класса


| Метод | Описание | С версии |
| --- | --- | --- |
| Init | Инициализирует интеграцию. |  |
| EnableSocnet | Активирует/деактивирует интеграцию. |  |
| IsEnabledSocnet | Проверяет активирована ли интеграция. |  |
| IsSocNet | Проверяет находится ли модуль в режиме интеграции. |  |

# CWiki

**CWiki** - Класс для работы c Wiki.


#### Методы класса


| Метод | Описание | С версии |
| --- | --- | --- |
| GetElementById | Получает Wiki-страницу по идентификатору. |  |
| GetElementByName | Получает Wiki-страницу по названию. |  |
| Add | Добавляет новую Wiki-страницу. |  |
| Update | Изменяет существующую Wiki-страницу. |  |
| Rename | Переименовывает Wiki-страницу. |  |
| Delete | Удаляет существующую Wiki-страницу. |  |
| GetDefaultPage | Получает Wiki-страницу по-умолчанию. |  |
| SetDefaultPage | Устанавливает Wiki-страницу по-умолчанию. |  |
| Recover | Восстанавливает страницу из истории. |  |
| UpdateHistory | Добавляет страницу в историю. |  |
| GetCategory | Получает список категорий. |  |
| UpdateCategory | Обновляет привязки страницы к категориям. |  |
| AddImage | Добавляет изображение к Wiki-странице. |  |
| DeleteImage | Удаляет изображение из Wiki-страницы. |  |

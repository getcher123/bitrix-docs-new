# CWikiUtils

**CWikiUtils** - Класс дополнительных функций.


#### Методы класса


| Метод | Описание | С версии |
| --- | --- | --- |
| getRightsLink | Возвращает массив ссылок для меню. |  |
| IsReadable | Проверяет доступ пользователя на чтение. |  |
| IsWriteable | Проверяет доступ пользователя на запись. |  |
| IsDeleteable | Проверяет доступ пользователя на удаление. |  |
| IsAllowHTML | Проверяет доступность пользователю использования визуального редактора. |  |
| CheckAccess | Проверяет доступ пользователя. | 9.5.3 |
| CheckServicePage | Проверяет является ли запрашиваемая страница сервисной. |  |
| IsCategoryPage | Проверяет является ли запрашиваемая страница категорией. |  |

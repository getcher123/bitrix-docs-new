## Важно про формат

В репозитории используются элементы Diplodoc/YFM (например, `_includes`, YAML‑файлы, расширенная разметка). В Obsidian/VS Code это может отображаться как обычный текст — используйте поиск по vault и переход по локальным ссылкам.

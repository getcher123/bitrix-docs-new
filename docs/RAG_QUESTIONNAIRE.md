# Опросник для старта (автономно) с предполагаемыми ответами

Ниже вопросы и черновые ответы. Исправь ответы прямо в этом файле.

## Цели и аудитория

**Вопрос:** Какова цель проекта и ключевой результат?  
**Предполагаемый ответ:** Быстрый локальный поиск/ответы по Bitrix‑документации с обязательными ссылками на `docs/...`.

**Вопрос:** Кто основные пользователи?  
**Предполагаемый ответ:** Разработчики и администраторы Bitrix.

**Вопрос:** Какие типовые запросы важнее всего?  
**Предполагаемый ответ:** Методы API, настройки в админке, шаги из курсов, REST‑методы.

## Источники и охват

**Вопрос:** Какие разделы vault приоритетны?  
**Предполагаемый ответ:** `docs/d7/`, `docs/bitrix24_api/`, `docs/courses/`, `docs/user_help/`, `docs/<module>/`.

**Вопрос:** Нужны ли внешние источники или только локальные файлы?  
**Предполагаемый ответ:** Только локальные файлы.

## Режим работы и инфраструктура

**Вопрос:** Работа строго offline?  
**Предполагаемый ответ:** Нет. Данные берём только локально из `docs/`, но допускаются сетевые вызовы к LLM и model endpoints (hybrid).

**Вопрос:** Доступные ресурсы (CPU/GPU/RAM/диск)?  
**Предполагаемый ответ:** CPU 4-8 ядер, RAM до 8 ГБ, диск 50+ ГБ, GPU не нужен.

**Вопрос:** Допустимы ли локальные модели (LLM/эмбеддинги)?  
**Предполагаемый ответ:** LLM: OpenAI `gpt-5`. Эмбеддинги и rerank: `bge-m3` и `bge-reranker-v2-m3` через эндпоинты (Google Colab).

**Вопрос:** Есть ли ограничения по установке ПО (Docker/conda)?  
**Предполагаемый ответ:** Docker допустим, можно ставить пакеты через pip или uv.

## Эндпоинты моделей (Google Colab)

**Вопрос:** Какой базовый URL эндпоинтов?  
**Предполагаемый ответ:** `https://xxxxxx.ngrok-free.app` (будет меняться при перезапуске Colab).

**Вопрос:** Есть ли авторизация на эндпоинтах (API key)?  
**Предполагаемый ответ:** Опционально. Если включаем — используем `X-API-Key`.

**Вопрос:** Какие таймауты и поведение при падении эндпоинтов?  
**Предполагаемый ответ:** Таймаут 30 сек, 3 ретрая; при недоступности — fallback на BM25‑only/без rerank.

## Стек и компоненты

**Вопрос:** Предпочитаемый стек?

**Вопрос:** Нужен ли UI или достаточно API/CLI?  
**Предполагаемый ответ:** На старте достаточно API и CLI.

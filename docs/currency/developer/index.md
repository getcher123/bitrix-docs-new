# Классы

Следующие классы используются для работы с модулем валют.


#### Классы


| Класс | Описание | С версии |
| --- | --- | --- |
| CCurrency | Класс для управления валютами: добавление, удаление, перечисление. | 3.3.2 |
| CCurrencyLang | Класс для работы с языкозависимыми параметрами валют (название, формат и пр.) | 3.3.2 |
| CCurrencyRates | Класс для работы с курсами валют: сохранение, конвертация и пр. | 3.3.2 |

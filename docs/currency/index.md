# currency — Валюты

Раздел содержит информацию для веб-разработчиков, использующих модуль **Валюты** для решения задач, выходящих за рамки возможностей стандартных [компонентов](https://dev.1c-bitrix.ru/user_help/settings/currency/components_2/index.php) модуля.

Перед использованием модуля необходимо проверить установлен ли он, и подключить его при помощи конструкции:


```
<?
if(CModule::IncludeModule("currency"))
{
	//здесь можно использовать функции и классы модуля
}
?>
```


Модуль включает в себя следующие Компоненты 2.0:

| **Компоненты модуля** |  |  |
| --- | --- | --- |
| **Название** | **Имя в пространстве имен bitrix** | **Описание** |
| **Таблица курсов валют** | **currency.rates** | Служит для отображения таблицы курсов валют в соответствии с их форматом. |

# Список классов


| Класс | Описание | С версии |
| --- | --- | --- |
| CTasks | Класс для получения списка задач и ряда других действий. | 10.0.2 по 12.0.9 |
| CTaskCheckListItem | Класс для работы с чеклистами. | 17.6.11 |
| CTaskElapsedItem | Класс для работы с учетом затраченного на задачи времени. Следует использовать вместо устаревшего CTaskElapsedTime. | 12.5.4 |
| CTaskElapsedTime | Класс для работы с учетом затраченного на задачи времени. | 10.0.5 |
| CTaskFiles | Класс для работы с файлами задач. | 10.0.5 |
| CTaskFilterCtrl | Класс для работы с предустановленными и пользовательскими фильтрами для задач. | 12.0.1 |
| CTaskItem | Класс для создания/редактирования/удаления и проведения некоторых других действий с задачами (строго рекомендуется к использованию вместо **CTasks**). | 12.5.0 |
| CTaskLog | Класс для работы с историей изменений в задаче. | 10.0.5 |
| CTaskReminders | Класс управления напоминаниями. | 10.0.6 |
| CTaskTags | Класс для работы с тегами. | 10.0.5 |

### Импорт каталога из CSV или XML-файла

Импортировать каталог товаров в систему можно из заранее подготовленного (предоставленного администратором) CSV или XML-файла. Импорт в формате XML обычно используется при интеграции с существующими системами учета, а файл CSV формата подготовливается вручную и данные импортируются (см.  [Импорт из файла CSV](/learning/course/index.php?COURSE_ID=42&CHAPTER_ID=05222)).

**Важно!** Необходимые права доступа пользователя к модулю **Торговый каталог** не ниже, чем **Управление импортом/экспортом**.

**Важно:** Сохранять файл импорта нужно в кодировке сайта, в который он будет импортирован. Чаще всего это **UTF-8**. Если в результате выполнения импорта у вас, например, не отображается текст на русском языке, то значит проблемы в кодировке файла.

Процедура импорта выполняется со страницы Магазин &gt; Настройки &gt; Импорт данных с помощью необходимого профиля импорта:

![Нажмите на рисунок, чтобы увеличить](../../../images/courses/34/dev.1c-bitrix.ru/images/content_manager/examples/catalog/products_import_sm.png)

**Примечание:** подробную информацию по выполнению ручного импорта из файла и созданию собственных профилей импорта можно посмотреть:

- в [документации по продукту](https://dev.1c-bitrix.ru/user_help/store/sale/settings/import/index.php);
- в специальном разделе курса [Администратор. Бизнес](https://dev.1c-bitrix.ru/learning/course/index.php?COURSE_ID=42&CHAPTER_ID=03457&LESSON_PATH=3912.4580.3457).

Для этого в первую очередь выполняются настройки со стороны *1С-Битрикс: Управление сайтом* (см. форму [Интеграция с 1С](/learning/course/index.php?COURSE_ID=131&LESSON_ID=3392)) . Настройки выполняет администратор сайта или пользователь, наделенный специальными правами.

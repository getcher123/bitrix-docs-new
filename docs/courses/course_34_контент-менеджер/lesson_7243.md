Делайте информацию в инфоблоках привлекательной, загружайте большое количество фотографий. При этом не нужно устанавливать специальное программное обеспечение, в инфоблоки встроен собственный редактор изображений.



### Видеоурок

**Примечание:** инструмент загрузки и обработки изображений доступен с версии 15.5.1. Его можно использовать для

			картинок анонса

                    ![](../../../images/courses/48/dev.1c-bitrix.ru/images/content_manager/iblocks/image_an.png)

		,

			детальных картинок

                    ![](../../../images/courses/48/dev.1c-bitrix.ru/images/content_manager/iblocks/image_detail.png)

		 элемента, для фотографий,

			загружаемых в свойство

                    ![](../../../images/courses/48/dev.1c-bitrix.ru/images/content_manager/iblocks/image_file.png)

		 типа **Файл** и для

			изображений

                    ![](../../../images/courses/48/dev.1c-bitrix.ru/images/content_manager/iblocks/image_section.png)

		 раздела.

### Как пользоваться редактором?

1. Откройте форму редактирования элемента (раздела).
2. Перетащите в область картинок
                      В области картинок показываются либо эскизы изображений, либо иконки с названием файлов.
  Это определяется нажатием кнопок 1 и 2 соответственно.
  ![](../../../images/courses/48/dev.1c-bitrix.ru/images/content_manager/iblocks/el_image1_sm.png)
  		 изображения с локального компьютера.
  Хотите загрузить из других источников? Используйте кнопку ![](../../../images/courses/48/dev.1c-bitrix.ru/images/content_manager/iblocks/but_other.png).
3. Дважды кликните по добавленному изображению либо нажмите на "карандашик" под изображением. Откроется окно редактора:
  ![Нажмите на рисунок, чтобы увеличить](../../../images/courses/48/dev.1c-bitrix.ru/images/content_manager/iblocks/el_image2_sm.png)
  **Примечание:** для добавляемых изображений редактор откроется автоматически, если предварительно в меню кнопки ![](../../../images/courses/48/dev.1c-bitrix.ru/images/content_manager/iblocks/but_other.png) отметить пункт
  			Использовать кадрирование
                      ![](../../../images/courses/48/dev.1c-bitrix.ru/images/admin_start/iblock/image_use_crop.png)
  		.
  Придайте подходящий вид изображению, отображаемому в рабочей области 1. При этом:
  - Инструменты панели 3 используйте для поворота, отражения,
    			обрезки изображения
                        При выполнении обрезки отображается специальная панель управления. Так, кнопка ![](../../../images/courses/48/dev.1c-bitrix.ru/images/admin_start/iblock/crop_del.png) позволяет очистить выделенную область, предназначенную для обрезки. Кнопка ![](../../../images/courses/48/dev.1c-bitrix.ru/images/admin_start/iblock/crop_save.png) служит непосредственно для обрезки изображения согласно выбранной области, а кнопка ![](../../../images/courses/48/dev.1c-bitrix.ru/images/admin_start/iblock/crop_ratio.png) определяет сохранять ли соотношение сторон при выборе обрезаемой области.
    		, а также при переводе изображения в черно-белый вид и добавлении подписи.
  - Сохраняйте настройки выполняемой вами обрезки по кнопке 5, чтобы применять их в дальнейшем.
  - Обрезайте изображение по вашим настройкам: в панели 4 выберите пресет и нажмите кнопку ![](../../../images/courses/48/dev.1c-bitrix.ru/images/admin_start/iblock/preset_cut.png). В рабочей области 1 сразу отобразится рамка обрезки нужного размера.
    Старые пресеты редактируйте по кнопке ![](../../../images/courses/48/dev.1c-bitrix.ru/images/admin_start/iblock/preset_edit.png) панели 4.
  - Изменяйте размеры изображения в абсолютном 6 или процентом соотношении 7.
  - Не бойтесь ошибиться. Любое действие отменяется по кнопке 8.
  Если вы загрузили несколько изображений в область, то не закрывайте редактор. Переключайтесь между обрабатываемыми изображениями с помощью панели 2.
  Сохраните выполненные настройки.

### Заключение

Вы узнали, как пользоваться встроенным инструментом загрузки и обработки изображений. Попробуйте инструмент в работе, добавьте изображения в элементы инфоблока на вашем сайте.

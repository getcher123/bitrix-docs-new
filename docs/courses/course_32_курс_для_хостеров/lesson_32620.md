### Установка




Скачайте дистрибутив [РОСА «ФРЕШ» 12.5.1](https://rosa.ru/rosa-linux-download-links/) с официального сайта и установите его.




Все последующие шаги описаны для версии Server.




В ОС РОСА применяется система управления программными пакетами `DNF`. С ее помощью установите обновления до последней версии. Отключите SELinux и перезапустите систему:




```
sudo -i
dnf update
echo 'SELINUX=disabled' > /etc/sysconfig/selinux
reboot
```





### Настройка портов




Обязательно откройте порты:




- 22 — ssh доступ,
- 80 / 443 — http / https web-сервер.




Остальные порты для ntlm, сервера мгновенных сообщений нужно открыть, если только они используются. Можно выбрать произвольные порты, а можно те, что используются в [CentOS](https://dev.1c-bitrix.ru/learning/course/index.php?COURSE_ID=37&LESSON_ID=8811):




- 8890 / 8891 — http/https ntlm,
- 8893 / 8894 — http/https сервер мгновенных сообщений.

По умолчанию система обновляет модули до последней доступной версии. Если у вас долгий цикл тестирования и обновления, это может привести к рассинхронизации версий тестового и боевого сайтов.




Используйте экспертный режим обновлений. Он позволяет:



- выбрать версии модулей для установки обновлений,
- экспортировать список версий и импортировать его на другой сайт.




Обновления важны для защиты сайта от угроз и утечки данных. Установка старых версий может оставить уязвимости в системе. Рекомендуем устанавливать самые свежие версии обновлений.




### Как включить режим Перейдите в Настройки > Настройки продукта > Настройки модулей > Главный модуль, вкладка Система обновлений. Отметьте Включить экспертный режим. Сохраните изменения. На странице Marketplace > Обновление платформы появится вкладка Экспертный режим. Как настроить установку обновлений Отметьте модули для обновления. В колонке Версия до выберите нужную версию из выпадающего списка. Система установит все обновления до выбранной версии. Перенос настроек. Настройки можно экспортировать в файл в формате JSON и импортировать на другой сайт. Например, перенести настройки с тестового сайта на боевой. Конфликты зависимостей. Система автоматически проверит зависимости между модулями и сообщит о конфликтах. Отметьте необходимые обновления, чтобы решить конфликт. Описания версий. По ссылке Подробности откроется список всех новых версий с описанием обновлений. Пример обновления модулей Задача: обновить модули iblock, sale, catalog. Предварительно проверить их на тестовом сайте. Настроить версии на тестовом сайте Включите экспертный режим. Выберите последние версии модулей iblock, catalog, sale для обновления: iblock 24.300.200 catalog 25.0.0 sale 25.0.100 Система проверит зависимости между модулями. Для установки catalog 25.0.0 необходимо наличие обновлений: ui 25.50.0. Для установки sale 25.0.0 необходимо наличие обновлений: seo 24.700.0. Отметьте для установки обновлений модули ui и seo c нужными версиями. Экспортируйте список версий в файл в формате JSON. Содержимое файла updates.json: { "iblock": "24.300.200", "catalog": "25.0.0", "sale": "25.0.100", "seo": "24.700.0", "ui": "25.50.0" } Нажмите кнопку Установить обновления. Проверить обновления Проверьте работу сайта после установки новых версий и переходите к переносу на боевой сайт. Перенести настройки на боевой сайт



1. Включите экспертный режим на боевом сайте.
2. Импортируйте файл `updates.json` с информацией об устанавливаемых версиях модулей. Появится сообщение *Версии обновлений успешно импортированы*.
3. Нажмите Установить обновления.

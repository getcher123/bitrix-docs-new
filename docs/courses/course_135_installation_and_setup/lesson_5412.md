### Выбор хостинга

Для размещения сайтов *на платформе «1С-Битрикс»* подходит любой хостинг, который соответствует [техническим требованиям продукта](lesson_2593.md).

В таблице ниже представлены данные для того, чтобы сориентироваться какой хостинг выбрать для оптимальной производительности в зависимости от [редакции продукта «1С-Битрикс»](http://www.1c-bitrix.ru/products/cms/editions/#tab-table-link).

### Скрипт bitrix_server_test

Компания «1С-Битрикс» рекомендует клиентам и разработчикам проверять потенциальный хостинг специальным скриптом [bitrix_server_test](https://dev.1c-bitrix.ru/download/scripts/bitrix_server_test.php). Скрипт постоянно обновляется под выявляемые проблемы и обновления системы.

Также можно выбрать хостинг для продуктов «1С-Битрикс» у наших сертифицированных [партнеров](http://www.1c-bitrix.ru/partners/hosting.php).

В самом продукте «1С-Битрикс» соответствие сервера требованиям системы можно протестировать модулем [Монитор производительности](/learning/course/index.php?COURSE_ID=32&CHAPTER_ID=1146).

#### Седьмой шаг установки (выбор решения)

![](../../../images/courses/135/dev.1c-bitrix.ru/images/admin_start/install/master/install_master_demo.png)

Выберите подходящее вам решение и запустите один из мастеров создания сайта:

- Корпоративный сайт производственной компании — решение на примере сайта мебельного производства, подходящее производственным организациям и компаниям для создания корпоративного проекта.
- Корпоративный сайт услуг — решение на примере сайта банка для организаций и компаний в сфере услуг для создания корпоративного проекта.
- [Интернет-магазин](lesson_5325.md) — решение на примере магазина одежды для тех, кто создает свою торговую площадку.
- Загрузить из Маркетплейс — найти другое подходящее решение в каталоге.

**Примечание:** При выборе Загрузить из Маркетплейс будет осуществлен переход к восьмому шагу  [Выбор модуля](lesson_2061.md) Мастера установки.

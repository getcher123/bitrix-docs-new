### Установка

Скачайте дистрибутив АЛЬТ 11 с [официального сайта](https://www.basealt.ru) и установите его.

Все последующие шаги описаны для сервера без дополнительных программных пакетов.

В Альт 11 применяется система управления программными пакетами `apt-get`. С ее помощью установите обновления до последней стабильной версии и перезапустите систему.

```
su -
apt-get update
apt-get dist-upgrade
reboot
```

### Настройка портов

Обязательно откройте порты:

- 22 — ssh доступ,
- 80 / 443 — http / https web-сервер.

Остальные порты для ntlm, сервера мгновенных сообщений нужно открыть, если только они используются. Можно выбрать произвольные порты, а можно те, что используются в [CentOS](https://dev.1c-bitrix.ru/learning/course/index.php?COURSE_ID=37&LESSON_ID=8811):

- 8890 / 8891 — http/https ntlm,
- 8893 / 8894 — http/https сервер мгновенных сообщений.

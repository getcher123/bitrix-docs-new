# Установка дистрибутива сайта в BitrixVM/BitrixEnv

**Навигация**
- [← Оглавление курса](index.md)
- [← Предыдущий: 1933 — Карта установки](lesson_1933.md)
- [Следующий: 4523 — Установка продукта с помощью BitrixSetup →](lesson_4523.md)

Официальная страница урока: https://dev.1c-bitrix.ru/learning/course/index.php?COURSE_ID=135&LESSON_ID=9369

Для установки продукта «1С-Битрикс»:

Введите в адресной строке браузера адрес

			**bitrix url**

                    ![](../../../images/courses/135/dev.1c-bitrix.ru/images/bitrixvm/vmbitrix7/start_vm/vmplayer5_1.png)

		, указанный в [BitrixVM](https://dev.1c-bitrix.ru/learning/course/index.php?COURSE_ID=37&LESSON_ID=29238) или [BitrixEnv](https://dev.1c-bitrix.ru/learning/course/index.php?COURSE_ID=37&LESSON_ID=29234). В конце добавьте к нему bitrixsetup.php, например, https://192.168.1.245/bitrixsetup.php. Откроется окно выбора продукта:

![Нажмите на рисунок, чтобы увеличить](../../../images/courses/135/dev.1c-bitrix.ru/images/bitrixvm/vmbitrix9/install_product/bitrixsetup_sm.png)

Выбирайте интересующий вас продукт. После загрузки запустится мастер установки продукта.

Подробнее рассказываем в учебном курсе [Виртуальная машина BitrixVM](https://dev.1c-bitrix.ru/learning/course/index.php?COURSE_ID=37&LESSON_ID=29244).

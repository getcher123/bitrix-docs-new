Меню **Действия** сервера распространяется только на текущий сервер, входящий в пул:




![](../../../images/courses/48/dev.1c-bitrix.ru/images/admin_expert/scale/server_menu.png)




Описание значений некоторых действий:




- **Удалить из пула** - позволяет
  			исключить сервер
                      Доступно только на дополнительных серверах, включенных в пул
  		 из пула.
- **Обновить Битрикс-окружение** и **Обновить систему** - команды, аналогичные
  			одноимённым Глобальным командам
                      В меню Глобальные действия собраны команды, чьё действие распространяется на все серверы, входящие в пул:
  [Подробнее ...](lesson_6724.md#env_update)
  		.
  **Внимание!** Обновления - сложная операция, в процессе которой происходит обновление системных файлов операционной системы сервера. Перед запуском обновления рекомендуется сделать полный бекап.
- Команды **Смены паролей** пользователей root или bitrix - позволяют сменить пароль пользователя.
  **Внимание!** При смене пароля суперпользователя **root** будьте внимательны!

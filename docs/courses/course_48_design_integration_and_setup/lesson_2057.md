Облегчайте поиск информации вместе с логическими операторами. Система допускает использование пяти видов логических операторов. Давайте их рассмотрим.





|  | #### Операторы языка запросов |
| --- | --- |




| \| **Оператор** \| **Синонимы** \| **Описание** \|<br>\| --- \| --- \| --- \|<br>\| **и** \| And, &, + \| Оператор логическое **и**. Подразумевается, что его можно опускать: запрос `свежая рыба` полностью эквивалентен запросу `свежая и рыба`. \|<br>\| **или** \| Or, &gt; \| Оператор логическое **или** позволяет искать элементы, содержащие хотя бы один из операндов. \|<br>\| **не** \| Not, ~ \| Оператор логическое **не** ограничивает поиск  элементами, не содержащими слово, указанное после оператора. \|<br>\| **( )** \|  \| Круглые скобки задают порядок действия логических операторов. При формировании строки запроса убедитесь, что для каждой открывающейся скобки есть парная закрывающаяся. \|<br>\| **""** \|  \| Поиск точной фразы. Обычно используется для поиска цитат. \| |
| --- |










|  | #### Примеры использования логических операторов |
| --- | --- |




- `свежая рыба или пылесос`
  По такому запросу находится информация, в которой встречаются либо слова "свежая" и "рыба", либо слово "пылесос".
- `свежая рыба не скумбрия`
  По такому запросу находится информация, в которой встречаются слова "свежая" и "рыба" и не встречается слово "скумбрия".
- `свежая рыба не (селедка или ставрида)`
  По такому запросу система найдет все страницы сайта, где упоминается "свежая рыба" любых сортов, кроме "селедки" или "ставриды".

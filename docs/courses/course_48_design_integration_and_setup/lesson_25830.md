|  | #### Назначение прав пользователям |
| --- | --- |





В первую очередь нужно настроить права доступа пользователей к модулю. По умолчанию только пользователям с полными правами доступен модуль. В модуле доступны две роли:



1. Роль **(Коннектор к Битрикс24) Администратор** предоставляет полные права на модуль.
2. Роль **(Коннектор к Битрикс24) Пользователь** предоставляет права на работу с модулем, но не дает возможности редактировать настройки.



Роли привязываются стандартными механизмами конфигураций *1С*.

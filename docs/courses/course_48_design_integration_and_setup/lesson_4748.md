Модуль **Опросы, голосования** позволяет организовывать различные голосования, опросы и публиковать их результаты на корпоративном портале.




Для работы в административном режиме с модулем **Опросы, голосования**, достаточно иметь права администратора портала. Для публикации опроса в публичном разделе нужно быть администратором системы.





## Пример создания опроса и изменения типа поля ответа

- Создать опрос в Ленте новостей может любой пользователь. По умолчанию тип поля ответа может быть установлен только
  **radio** — радио-кнопка, можно выбрать только один вариант ответа из группы ответов с таким типом поля или **checkbox** — флаг, можно выбрать несколько вариантов ответов
- Изменить тип поля может только Администратор системы
- Делается это в Административном разделе в меню Сервисы &gt; Опросы &gt; Список опросов
- На закладке Ответы можно выбрать тип поля, например, **textarea** — многострочное поле ввода, которое посетитель сможет заполнить собственным ответом
- Администратор системы может посмотреть результаты опроса в Административном разделе. И выбрать, в каком виде будут представлены результаты — в виде гистограммы, таблицы или диаграммы
- Если опрос запустил пользователь, не обладающий правами Администратора системы, то результаты он сможет посмотреть в виде таблицы Excel после остановки опроса

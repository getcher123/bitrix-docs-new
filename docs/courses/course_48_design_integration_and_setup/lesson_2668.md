|  | ### Немного теории: уровни Проактивной защиты |
| --- | --- |

Любой веб-проект на базе *Bitrix Framework* обязательно имеет **начальный** уровень защиты. Повысить его можно с помощью модуля **Проактивная защита**, настроив один из следующих уровней безопасности:

- **Стандартный**;
- **Высокий**;
- **Повышенный**.



Причем для уровней справедливо понятие «вложенности»: чтобы настроить защиту сайта на высоком уровне, необходимо сначала настроить стандартный, а затем настроить все параметры высокого уровня. Соответственно, чтобы настроить защиту на повышенном уровне, необходимо настроить высокий, а затем настроить параметры повышенного уровня.

Информация о текущем уровне безопасности сайта представлена на странице **Панель безопасности** (Настройки &gt; Проактивная защита &gt; Панель безопасности).

Ниже для каждого уровня приведена соответствующая таблица параметров и их значений, а также указаны рекомендации по изменению значений параметров (то есть если для какого-то параметра установлено несоответствующее значение, то в поле **Рекомендации** будет отображено необходимое для выполнения действие).

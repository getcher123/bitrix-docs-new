Настройки для выгрузки данных в живую ленту расположены в административной части портала в разделе **Сервисы** (Сервисы &gt; Внешние источники данных &gt; Публикация в Живую ленту). Всего с порталом может работать неограниченное количество внешних источников.




![](../../../images/courses/48/dev.1c-bitrix.ru/images/portal_admin/admin_sys/xdimport/v12/import_settings1_sm.png)




Для выгрузки данных из 1С в живую ленту используется режим **POST**.




**Примечание:** подробное описание настроек для режима POST представлено в уроке [Публикация в Живую ленту](lesson_3463.md#post).

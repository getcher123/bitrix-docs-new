Создавать и редактировать шаблоны бизнес-процессов (БП) в административном разделе сайта можно почти для всех инфоблоков. Исключение составляют инфоблоки, которые используются в рамках **Библиотеки документов** и **Бизнес-процессов организаций**.




Шаблоны БП для модуля **Библиотека документов** нужно создавать и редактировать только через публичный раздел сайта. Такое ограничение связано с особенностями работы с документами портала. Шаблоны, созданные через административный раздел, не будут доступны в публичной части сайта и могут использоваться только в административном разделе.




В [следующем уроке](lesson_3816.md) вы узнаете, как настраивать параметры шаблона бизнес-процессов.

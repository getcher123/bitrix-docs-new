Действие передаёт информацию в другие системы с помощью

			Вебхука

                    **WebHook** — упрощенный вариант rest-событий и rest-команд, без написания приложения. Для коробочных версий доступен только при наличии модуля Rest.

		. Действие не возвращает никаких данных, оно только отправляет запрос.






#### Параметры действия




- **Хендлер** — URL, который нужно использовать при отправке данных из *Битрикс24*.






**Примечание.** Это действие предназначено для разработчиков. Если вы не знаете, что такое Вебхук, пропустите этот урок.

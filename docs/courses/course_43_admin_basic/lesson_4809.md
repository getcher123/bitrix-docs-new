[Структура файлов и папок](lesson_2823.md) модулей на  D7 осталась такой же, как и в старой версии ядра.

API (классы, логика) модуля располагается в подпапке `/lib

                    Данная папка не является обязательной, если у вашего класса нет собственных методов.` папки модуля. Например, для Главного модуля путь будет таким: `bitrix/modules/main/lib`.  Из этой же папки API будет автоматически подключаться при соблюдении некоторых правил:

- Файлы классов должны именоваться в нижнем регистре, при этом название класса равно название файла. Пример: `/lib/myclass.php`.
- Файлы должны содержать правильный namespace. Пример: если модуль подключается как:
  `Loader::includeModule('company.shotname');`
  то в классе должен быть прописан namespace: **namespace Company\Shotname**.
- Там где классы модуля используются в административной части - модуль должен быть подключен.

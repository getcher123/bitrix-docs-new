# Магазин

Источник: https://dev.1c-bitrix.ru/user_help/store/index.php

| **Модули раздела** |  |
| --- | --- |
| **Название** | **Описание** |
| [Мастер магазина](https://dev.1c-bitrix.ru/user_help/store/storeassist.php) | Инструмент помощи для владельца магазина в настройке и подготовке магазина к работе. |
| [Интернет-магазин](https://dev.1c-bitrix.ru/user_help/store/sale/index.php) | Позволяет осуществлять продажу товаров и услуг с сайта. Интернет-магазин состоит из следующих функциональных элементов:<br><br>- **корзина товаров и услуг** с возможностью добавления туда произвольных товаров и услуг;<br>- процедура **оформления заказа**;<br>- **персональный раздел покупателя** с возможностью просмотра сделанных заказов и настройкой профилей;<br>- **административный раздел** для настройки параметров магазина и управления заказами клиентов. |
| [Торговый каталог](https://dev.1c-bitrix.ru/user_help/store/catalog/index.php) | Модуль системы **"1С-Битрикс: Управление сайтом"**, позволяющий:<br><br>- создавать торговые каталоги товаров, услуг и всевозможного цифрового контента;<br>- осуществлять продажу товаров, услуг и цифрового контента через интернет-магазин;<br>- осуществлять продажу прав доступа к ресурсам сайта;<br>- управлять ценами на товары и услуги торговых каталогов;<br>- загружать и выгружать данные о товарах, заказах, продажах в форматах 	**CommerceML**, **Froogle**, **Yandex** и **CSV** (формат последнего можно настраивать при загрузке);<br>- загружать и выгружать данные в любой из форматов в соответствии с созданными пользователем профилями. |

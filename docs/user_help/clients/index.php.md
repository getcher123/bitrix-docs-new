# Клиенты

Источник: https://dev.1c-bitrix.ru/user_help/clients/index.php

| **Сущности раздела** |  |
| --- | --- |
| **Название** | **Описание** |
| [Коммуникации с клиентами](https://dev.1c-bitrix.ru/user_help/clients/clientscomm.php) | Модуль решает следующие задачи:<br>  <br>- Гибкая настройка необходимых каналов связи для каждой части сайта;<br>- Настройка сценариев работы для мобильных устройств;<br>- Информационная поддержка клиентов;<br>- Полная интеграция с Битрикс24.CRM;<br>- Динамическое управление потоком клиентов (от первого входа до повторных продаж);<br>- Ведение единой клиентской базы для работы нескольких сайтов. |
| [Настройки модуля](https://dev.1c-bitrix.ru/user_help/clients/settings.php) | Форма настроек модуля **Интеграция с Битрикс24** (*Настройки > Настройки продукта > Настройки модулей > Интеграция с Битрикс24*) предназначена для подключения портала Битрикс24 к сайту и разделения прав на доступ к модулю. |

# Основные сведения

Источник: https://dev.1c-bitrix.ru/user_help/help/index.php

Справочная система по **Bitrix Framework** состоит из  [API](http://dev.1c-bitrix.ru/api_help), [API D7](https://dev.1c-bitrix.ru/api_d7/), [Rest API](https://dev.1c-bitrix.ru/rest_help/index.php) и пользовательской (та, которую вы в данный момент читаете) документаций.

API документация предназначена для разработчиков и содержит описание принципов кастомизации CMS, описание классов, методов, событий, констант. REST API документация предназначена для создания собственных приложений для облачной реализации Битрикс24. Пользовательская документация содержит описание компонентов системы и описание полей настроек административной части системы.

Все виды документации предназначены только для получения **справочной информации** и не содержат описания принципов и сценариев работы с системой. Для изучения принципов и сценариев обратитесь к Учебным курсам.

Учебные курсы построены по принципу "от пользователя". То есть разбиты по предполагаемым ролям, которые выполняются пользователями системы:

- [Контент-менеджер](https://dev.1c-bitrix.ru/learning/course/index.php?COURSE_ID=34). (Описание интерфейсов и принципов работы системы для Контент-менеджера, то есть тех, кто работает с контентом уже готовых сайтов.)
- [Тесты на сертификат "Пользователь сервиса Битрикс24"](https://dev.1c-bitrix.ru/learning/course/index.php?COURSE_ID=52).
- [Тесты на сертификат "Пользователь коробочной версии Битрикс24"](https://dev.1c-bitrix.ru/learning/course/index.php?COURSE_ID=71).
- Администратор [Базовый](https://dev.1c-bitrix.ru/learning/course/index.php?COURSE_ID=35) (Основные принципы администрирования в системе), [Модули](https://dev.1c-bitrix.ru/learning/course/index.php?COURSE_ID=41) (Описание работы с большинством модулей системы.), [Бизнес](https://dev.1c-bitrix.ru/learning/course/index.php?COURSE_ID=42) (Описание работы с модулями системы коммерческой направленности.).
- [Администратор КП](https://dev.1c-bitrix.ru/learning/course/index.php?COURSE_ID=48). (Администрирование коробочной версии Битрикс24)
- [Разработчик Bitrix Framework](https://dev.1c-bitrix.ru/learning/course/index.php?COURSE_ID=43). Описание принципов разработки проектов на **Bitrix Framework**.
- [Курс для хостеров](https://dev.1c-bitrix.ru/learning/course/index.php?COURSE_ID=32). Описание принципов настроек хостинга под **Bitrix Framework**
- [Технология Композитный сайт](https://dev.1c-bitrix.ru/learning/course/index.php?COURSE_ID=39). Описание технологии и принципов создания сайтов с её использованием.
- [Разработка и эксплуатация высоконагруженных проектов](https://dev.1c-bitrix.ru/learning/course/index.php?COURSE_ID=38).

Для облегчения изучения системы можно также использовать [видеоролики](http://dev.1c-bitrix.ru/learning/video.php). Следить за изменениями в документации можно с помощью страницы [Что нового?](https://dev.1c-bitrix.ru/docs/what-s-new.php), подписавшись на канал в [Телеграмме](https://t.me/bitrixdoc), а также в блоге [Отдела документации](https://dev.1c-bitrix.ru/community/blogs/Docs_and_other/).

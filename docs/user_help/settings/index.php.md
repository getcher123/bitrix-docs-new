# Настройки

Источник: https://dev.1c-bitrix.ru/user_help/settings/index.php

| **Сущности раздела** |  |
| --- | --- |
| **Название** | **Описание** |
| [Избранное](https://dev.1c-bitrix.ru/user_help/settings/favorites/index.php) | Служит для быстрого перехода к заранее составленному списку страниц. |
| [Управление пользователями](https://dev.1c-bitrix.ru/user_help/settings/users/index.php) | Позволяет создавать произвольное количество групп пользователей с разными правами доступа к ресурсам сайта. |
| [Поиск](https://dev.1c-bitrix.ru/user_help/settings/search/index.php) | Осуществляет индексирование и поиск информации на сайте. Вся текстовая информация, которая попадает на сайт в виде статических HTML страниц или через модули (например, модуль **Информационные блоки**, модуль **Форум** и т.д.) автоматически индексируется. Поиск может осуществляться как по всей проиндексированной информации, так и по ее отдельным частям. |
| [Проактивная защита](https://dev.1c-bitrix.ru/user_help/settings/security/index.php) | Является важным дополнением к стандартной политике безопасности продукта и существенно повышает уровень защиты веб-проекта. |
| DAV (КП) | Предназначен для обмена календарями и контактами с внешними системами. |
| [Push and Pull](https://dev.1c-bitrix.ru/user_help/settings/pull/index.php) | Позволяет организовать транспорт мгновенных нотификаций и сообщений клиентам. |
| [REST API](https://dev.1c-bitrix.ru/user_help/settings/rest_api/settings.php) | Используется для разработки собственных приложений или интеграций для Битрикс. |
| [Валюты](https://dev.1c-bitrix.ru/user_help/settings/currency/index.php) | Необходим для работы модулей **Торговый каталог** и **Интернет-магазин**, позволяет управлять валютами и их курсами. |
| [Веб-кластер](https://dev.1c-bitrix.ru/user_help/settings/cluster/index.php) | Это комбинация технологических решений, которые позволяют распределить один сайт на несколько серверов, решая тем самым несколько задач: обеспечение высокой доступности сайта; его масштабирование в условиях возрастающей нагрузки; балансирование нагрузки, трафика, данных между несколькими серверами. |
| [Веб-сервисы](https://dev.1c-bitrix.ru/user_help/settings/webservice/index.php) | С помощью модуля **Веб-сервисы** можно организовать систему веб-сервисов и SOAP. |
| [Коннекторы для внешних мессенджеров (КП)](https://dev.1c-bitrix.ru/user_help/settings/imconnector/index.php) | Выполняет подключение каналов открытых линий. |
| [Открытые линии (КП)](https://dev.1c-bitrix.ru/user_help/settings/imopenlines/index.php) | Позволяет организовать подключение различных внешних социальных сервисов для общения с помощью открытых линий. |
| [Телефония](https://dev.1c-bitrix.ru/user_help/settings/voximplant/index.php) | Предназначен для совершения входящих и исходящих звонков на вашем Битрикс24. |
| [Облако 1С-Битрикс](https://dev.1c-bitrix.ru/user_help/settings/bitrixcloud/index.php) | Раздел административной части **Облако 1С-Битрикс** содержит набор облачных сервисов и позволяет:<br><br>- включить и настроить поддержку CDN;<br>- включить и настроить инсектирование работоспособности сайта;<br>- создавать и просматривать резервные копии, которые хранятся в облаке. |
| [Облачные хранилища](https://dev.1c-bitrix.ru/user_help/settings/clouds/index.php) | Предназначен для подключения к проекту поддержки "облачных" хранилищ файлов. |
| [AD/LDAP интеграция](https://dev.1c-bitrix.ru/user_help/settings/ldap/index.php) | Позволяет интегрировать продукт **"1С-Битрикс: Управление сайтом"** с корпоративными сетями, построенными с использованием **Microsoft Windows Server 2000/2003/2008**, **Unix** и **Novell**. |
| [Перевод](translate/index.php.md) | Служит для удобства перевода файлов сообщений (заголовки таблиц, ссылки, текст кнопок и т.п.) на нескольких языков. |
| [XMPP сервер (КП)](https://dev.1c-bitrix.ru/user_help/settings/xmpp/index.php) | Предназначен для организации общения и учета присутствия сотрудников корпоративного портала, использующих **Jabber**-клиенты. Это можно делать только на выделенном сервере или на виртуальном выделенном сервере (**VPS**). |
| [Настройки продукта (главный модуль)](https://dev.1c-bitrix.ru/user_help/settings/settings/index.php) | Обеспечивает общее функционирование системы, взаимодействие всех модулей продукта и позволяет создавать, поддерживать и управлять неограниченным числом сайтов. |
| Инструменты |  |
| [Монитор производительности](https://dev.1c-bitrix.ru/user_help/settings/perfmon/index.php) | Предназначен для мониторинга параметров производительности сайта. |
| [ЧатБоты (КП)](https://dev.1c-bitrix.ru/user_help/settings/imbot/index.php) | Модуль управляет чат-ботами *Битрикс24*. Установка модуля не происходит автоматически, так же необходимо будет осуществить настройку локальной сети для доступа к порталу из сети. |
| [Компрессия](https://dev.1c-bitrix.ru/user_help/settings/compression/index.php) | Осуществляет сжатие страниц для ускорения вывода содержания сайта пользователям. Модуль в несколько раз уменьшает объем передаваемых HTML-данных между сайтом и браузером клиента, что существенно увеличивает скорость работы как для посетителей, так и для администраторов сайта. |

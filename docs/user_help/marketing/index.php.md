# Маркетинг

Источник: https://dev.1c-bitrix.ru/user_help/marketing/index.php

### Функционал раздела

| **Функционал раздела** |  |
| --- | --- |
| **Название** | **Описание** |
| [Пульс конверсии](https://dev.1c-bitrix.ru/user_help/marketing/conversion_pulse.php) | Конверсия - ключевой показатель эффективности работы сайта, который показывает процентное отношение числа посетителей сайта, выполнивших на нём какие-либо целевые действия, к общему числу посетителей сайта. |
| [Бизнес модель интернет-магазина](https://dev.1c-bitrix.ru/user_help/marketing/web_store_business_model.php) | Инструмент прогнозирования эффективности бизнес-модели интернет-магазина. |
| [A/B-тестирование](https://dev.1c-bitrix.ru/user_help/marketing/ab_testing/index.php) | A/B-тестирование - один из эффективных методов маркетингового исследования, суть которого в сравнении двух представлений выбранной страницы или дизайна сайта, для того чтобы выяснить насколько изменения повысят конверсию. |
| [Автоконтекст](https://dev.1c-bitrix.ru/user_help/marketing/context_adv/index.php) | Предназначен для взаимодействия и управления системой размещения поисковой и тематической рекламы Яндекс.Директ |
| [Торговый маркетинг](https://dev.1c-bitrix.ru/user_help/marketing/discounts/index.php) | Предназначен для создания акций и скидок в интернет-магазине. |
| [Email-маркетинг](https://dev.1c-bitrix.ru/user_help/marketing/sender/index.php) | Служит для организации работы рассылок. |
| [Реклама](https://dev.1c-bitrix.ru/user_help/marketing/ads.php) | Сервис создания рекламы для нужного сегмента аудитории в поисковых сервисах и социальных сетях. |
| [Триггерные рассылки](https://dev.1c-bitrix.ru/user_help/marketing/triggered_emails/index.php) | Триггерные рассылки относятся к модулю Email-маркетинг. Функционал предназначен для автоматической отправки сайтом последовательности писем в ответ на действие пользователя или после наступления некоторого события, связанного с пользователем. |
| Персонализация | Облачный сервис персонализации 1С-Битрикс BigData, сохраняя полную анонимность данных, анализирует посетителей, зашедших на ваш сайт (их интересы, покупки, сходство с другими покупателями), и составляет для них персональные рекомендации. |
| [Поисковая оптимизация](https://dev.1c-bitrix.ru/user_help/marketing/seo/index.php) | Предназначен для анализа технического исполнения сайта. Он отображает параметры, влияющие на оптимизацию и дает рекомендации по улучшению этих параметров. |
| [Баннерная реклама](https://dev.1c-bitrix.ru/user_help/marketing/advertising/index.php) | Система управления рекламой позволяет организовать показ баннеров на сайте, подключив код внешней баннерной системы, либо за счет внутренней ротации баннеров в пределах сайта. |

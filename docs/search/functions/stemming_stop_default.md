# Stemming_stop_default


```
bool
stemming_stop_default(
	string sWord
);
```

Функция определения стоп слова применяемая по умолчанию.


#### Параметры функции


| Параметр | Описание |
| --- | --- |
| sWord | Неизменная основа слова. |


#### Возвращаемое значение

Функция возвращает true если sWord не является стоп словом и false в противном случае. Определение происходит по длине слова. Стоп словами считаются все, имеющие длину меньше двух символов.

# События модуля


| Событие | Описание | С версии |
| --- | --- | --- |
| [BeforeIndex](beforeindex.md) | Вызывается перед индексацией элемента. | 5.1.0 |
| [OnSearch](onsearch.md) | Вызывается перед выполнением поисковых запросов. | 3.0.4 |
| [OnReIndex](onreindex.md) | Вызывается при построении поискового индекса. | 3.0.4 |
| [OnSearchGetFileContent](onsearchgetfilecontent.md) | Вызывается при индексации статических файлов. | 6.5.6 |
| [OnSearchGetURL](onsearchgeturl.md) | Вызывается при форматировании элемента в результатах поиска. | 4.0.0 |
| [OnSearchGetTag](onsearchgettag.md) | Вызывается при разборе тегов. | 7.1.2 |
| [OnBeforeFullReindexClear](onbeforefullreindexclear.md) | Вызывается в начале первого шага полной переиндексации, непосредственно перед удалением всех данных поискового индекса. | 8.0.4 |
| [OnBeforeIndexDelete](onbeforeindexdelete.md) | Вызывается перед удалением части поискового индекса. | 8.0.4 |
| [OnBeforeIndexUpdate](onbeforeindexupdate.md) | Вызывается перед обновлением поискового индекса. | 8.0.4 |
| [OnAfterIndexAdd](onafterindexadd.md) | Вызывается после добавления новых данных в поисковый индекс. | 8.0.4 |
| [OnSearchCheckPermissions](onsearchcheckpermissions.md) | Вызывается при построении поискового запроса. | 8.0.4 |

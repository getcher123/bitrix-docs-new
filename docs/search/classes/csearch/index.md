# Класс CSearch

Класс для индексирования сайта и осуществления поиска по индексу


#### Методы класса


| Метод | Описание | С версии |
| --- | --- | --- |
| CSearch | Конструктор класса CSearch | 3.0.4 |
| Search | Возвращает записи индекса, которые удовлетворяют заданной строке запроса и указанным параметрам, а так же доступны на просмотр для текущего посетителя (в соответствии с его уровнем доступа) | 3.3.2 |
| ChangeIndex | Метод изменяет проиндексированную информацию данного модуля | 3.0.4 |
| ChangePermission | Метод изменяет права на доступ к проиндексированной информации данного модуля | 3.0.4 |
| ChangeSite | Метод изменяет привязку проиндексированной информации к сайтам | 8.0.4 |
| DeleteIndex | Метод удаляет из индекса поиска указанную информацию | 3.0.4 |
| Index | Метод переиндексирует какую-то одиночную позицию (сообщение на форуме, новость и т.п.), причем комбинация (MODULE_ID, ITEM_ID) используется для определения переиндексируемого документа | 3.0.1 |
| ReindexModule | Метод инициирует переиндексацию указанного модуля | 3.0.4 |
| ReIndexAll | Пошаговый метод переиндексации | 3.0.4 |
| ReIndexFile | Переиндексация статического файла | 3.0.1 |
| Fetch | Возвращает следующий найденный элемент | 4.0.0 |
| SetOptions | Устанавливает дополнительные опции поиска |  |

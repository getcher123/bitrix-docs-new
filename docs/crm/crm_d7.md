# CRM в ядре D7

Список классов модуля **CRM** в ядре D7.


| Класс, пространство имён | Описание |
| --- | --- |
| **Activity** | Пространство имён содержит классы для работы с делами в CRM. |
| **Automation** | Пространство имён cодержит классы для автоматизации продаж в CRM. |
| **BankDetailTable** | Класс предназначен для работы с банковскими реквизитами. |
| **Binding** | Пространство имён классов для множественных привязок сущностей CRM. |
| **CallList** | Подпространство имён содержит классы для работы со звонками. |
| **Category** | Пространство имён классов для работы с Направлениями. |
| **Color** | Пространство имён классов для хранения цветовых схем стадий Сделки, статусов Лида и Предложения в CRM. |
| **CommunicationType** | класс для работы с типами связей. |
| **CompanyAddress** | Класс для хранения адресов Компаний. |
| **Config** | Пространство имён классов для хранения пользовательских настроек в контексте определенной сущности. |
| **ContactAddress** | Класс для хранения адресов Контактов. |
| **Conversion** | Пространство имён классов для конвертации сущностей CRM. |
| **Entity** | Пространство имён содержит классы для работы с сущностями. |
| **EntityAddress** | Класс для хранения адресов сущностей. |
| **EntityBankDetail** | Класс для работы с банковскими реквизитами. |
| **EntityRequisite** | Класс для работы с реквизитами сущности. |
| **Entry** | Пространство имён содержит классы для хранения отдельных записей. |
| **Integration** | Пространство имён классов для для взаимодействия с другими модулями - интеграции. |
| **Integrity** | Пространство имён содержит классы для контроля, поиска, отслеживания и объединения дубликатов. |
| **InvoiceTable** | Класс предназначен для работы с таблицей счетов. |
| **Kanban** | Пространство имён содержит классы для работы с канбаном. |
| **LeadAddress** | Класс для хранения адресов Лидов. |
| **Merger** | Пространство имён классов, используемых при управлении дубликатами, при конвертации и для объединения полей сущностей. |
| **PresetTable** | Класс ORM для работы с шаблонами реквизитов CRM. |
| **Preview** | Пространство имён классов, используемых при формировании "богатых ссылок" на сущности CRM. |
| **Requisite** | Пространство имён классов, используемых для конвертации реквизитов из сущностей. |
| **RequisiteAddress** | Класс для хранения адресов реквизитов. |
| **RequisiteTable** | Класс для работы с реквизитами сущностей. |
| **Restriction** | Пространство имён содержит классы для работы с ограничениями. |
| **Settings** | Пространство имён классов и методов для выполнения различных неосновных настроек системы CRM. |
| **Statistics** | Пространство имён классов, используемых для сбора данных для построения аналитических отчетов. |
| **Synchronization** | Пространство имён классов, используемых для синхронизации наборов пользовательских полей между различными типами сущностей. |
| **Widget** | Виджеты, используемые для построения аналитических отчётов. |

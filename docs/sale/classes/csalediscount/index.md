# Класс CSaleDiscount

Метод `CSaleDiscount::DoProcessOrder` использовался для расчета правил корзины до выхода нового ядра магазина. С версии 15.5 считается устаревшим и строго не рекомендованным к использованию. Вместо него используйте класс **\Bitrix\Sale\Discount**.


#### Методы класса


| Метод | Описание | С версии |
| --- | --- | --- |
| GetList | Метод возвращает результат выборки записей из скидок на заказ в соответствии со своими параметрами. | 3.3.1 |
| GetByID | Метод возвращает параметры скидки с кодом ID. | 3.3.1 |
| Update | Метод обновляет параметры скидки с кодом ID на параметры из массива arFields. | 3.3.1 |
| Delete | Метод удаляет скидку с кодом ID. | 3.3.1 |
| Add | Метод добавляет новую скидку на сумму заказа с параметрами из массива *arFields*. | 4.0.6 |

# Класс CSaleStatus


#### Методы класса


| Метод | Описание | С версии |
| --- | --- | --- |
| Add | Метод добавляет новый статус заказа с параметрами из массива arFields. | 3.3.0 |
| Update | Метод изменяет параметры статуса заказа с кодом ID. | 3.3.0 |
| Delete | Метод удаляет статус заказа с кодом ID. Если в базе есть заказы, находящиеся в этом статусе, то этот статус удалить нельзя. | 3.3.0 |
| GetList | Метод возвращает результат выборки записей из статусов в соответствии со своими параметрами. | 3.3.0 |
| GetByID | Метод возвращает параметры статуса с кодом ID, включая языкозависимые параметры для языка strLang. | 3.3.0 |

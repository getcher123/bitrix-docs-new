# CSaleAuxiliary


#### Методы класса


| Метод | Описание | С версии |
| --- | --- | --- |
| DeleteByTime | Метод удаляет всю информацию о временном доступе, которая старше указанного времени. | 4.0.10 |
| DeleteByUserID | Метод удаляет всю информацию о временном доступе пользователя с кодом userID. | 4.0.10 |
| GetByID | Метод выбирает параметры информации о временном доступе с кодом ID. | 4.0.10 |
| GetByParams | Метод выбирает параметры информации о временном доступе к ресурсу с идентификатором itemMD5 для пользователя с кодом userID. | 4.0.10 |
| GetList | Метод возвращает результат выборки записей информации о временном доступе в соответствии со своими параметрами. | 4.0.10 |
| Update | Метод изменяет параметры записи с кодом ID информации о временном доступе к ресурсу в соответствии с данными из массива arFields. | 4.0.10 |
| Add | Метод добавляет новую запись с информацией о временном доступе к ресурсу в соответствии с данными из массива arFields. | 4.0.10 |
| CheckAccess | Метод проверяет, может ли посетитель с кодом userID получить доступ к ресурсу с идентифицирующей строкой itemMD5. | 4.0.10 |
| Delete | Метод удаляет информацию о временном доступе с кодом ID. | 4.0.10 |

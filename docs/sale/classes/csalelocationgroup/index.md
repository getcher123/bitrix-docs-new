# Класс CSaleLocationGroup


#### Методы класса

Класс устарел. Рекомендуется использование класса `\Bitrix\Sale\Location\GroupTable`.


| Метод | Описание | С версии |
| --- | --- | --- |
| GetLocationList | Метод возвращает набор местоположений, связанных с группами местоположений, удовлетворяющих фильтру arFilter. | 3.2.2 |
| GetGroupLangByID | Метод возвращает языкозависимые параметры группы местоположений с кодом ID для языка с кодом strLang. | 3.2.2 |
| Update | Метод обновляет параметры группы местоположений с кодом ID на значения из массива arFields. | 3.2.2 |
| Delete | Метод удаляет группу местоположений с кодом ID. Местоположения, входящие в эту группу, не изменяются. | 3.2.2 |
| GetList | Метод возвращает набор групп местоположений, удовлетворяющих фильтру arFilter. Группы отсортированы в соответствии с массивом arOrder. | 3.3.0 |
| GetByID | Метод возвращает параметры группы местоположений с кодом ID. Языкозависимые параметры возвращаются для языка с кодом strLang. | 3.3.0 |
| Add | Метод добавляет новую группу местоположений с параметрами из массива arFields. | 3.3.0 |

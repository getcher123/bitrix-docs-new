# Класс CSaleRecurring


#### Методы класса


| Метод | Описание | С версии |
| --- | --- | --- |
| Update | Метод изменяет параметры записи на продление подписки в соответствии с параметрами из массива arFields. | 4.0.6 |
| Add | Метод добавляет новую запись на продление подписки в соответствии с параметрами из массива arFields. | 4.0.6 |
| Delete | Метод удаляет запись на продление подписки с кодом ID. | 4.0.6 |
| CheckRecurring | Метод ищет подписки, которые пора продлить, и пытается осуществить продление. | 4.0.6 |
| NextPayment | Метод осуществляет продление подписки с кодом ID. | 4.0.6 |
| CancelRecurring | Метод осуществляет отмену продления подписки с кодом ID. | 4.0.6 |
| GetList | Метод возвращает результат выборки записей информации и продлении подписки в соответствии со своими параметрами. | 4.0.6 |
| GetByID | Метод возвращает параметры информации о продлении с кодом ID. | 4.0.6 |

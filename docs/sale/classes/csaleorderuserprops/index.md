# Класс CSaleOrderUserProps


#### Методы класса


| Метод | Описание | С версии |
| --- | --- | --- |
| GetByID | Метод возвращает параметры профиля покупателя с кодом ID. | 3.3.0 |
| Update | Метод обновляет параметры профиля покупателя с кодом ID на значения из массива arFields. | 3.3.0 |
| Delete | Метод удаляет профиль покупателя с кодом ID. Вместе с профилем удаляются все его свойства. | 3.3.0 |
| Add | Метод сохраняет новый профиль покупателя с параметрами из массива arFields. | 3.3.5 |
| ClearEmpty | Метод удаляет все пустые профили из базы (т.е. профили, в которых нет свойств). | 3.3.0 |
| GetList | Метод возвращает результат выборки записей из профилей покупателя в соответствии со своими параметрами. | 3.3.0 |

# Класс CSalePersonType

| --- | --- | --- |
| GetList | Метод возвращает результат выборки записей из типов плательщика в соответствии со своими параметрами. | 3.3.0 |

| Update | Метод обновляет параметры типа плательщика с кодом ID. | 3.3.0 |
| Delete | Метод удаляет тип плательщика с кодом ID. | 3.3.0 |
| Add | Метод добавляет новый тип плательщика с параметрами из массива arFields. | 3.3.5 |

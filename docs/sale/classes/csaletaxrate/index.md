# Класс CSaleTaxRate


#### Методы класса


| Метод | Описание | С версии |
| --- | --- | --- |
| Update | Метод изменяет параметры ставки налога с кодом ID. | 3.3.1 |
| Delete | Метод удаляет ставку налога с кодом ID. | 3.3.1 |
| GetByID | Метод возвращает параметры ставки налога с кодом ID. | 3.3.1 |
| Add | Метод добавляет новую ставку налога с параметрами из массива arFields. | 3.3.1 |
| GetList | Метод возвращает набор ставок налога, удовлетворяющих фильтру arFilter. Набор упорядочен в соответствии с массивом arOrder. | 3.3.1 |

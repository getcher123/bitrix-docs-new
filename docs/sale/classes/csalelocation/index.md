# Класс CSaleLocation

Класс устарел. Рекомендуется использовать методы класса **Bitrix\Sale\Location**.


#### Методы класса


| Метод | Описание | С версии | Устарел с версии |
| --- | --- | --- | --- |
| GetCountryByID | Метод возвращает языконезависимые параметры страны с кодом ID. | 3.2.2 | 14.10 |
| GetCountryLangByID | Метод возвращает языкозависимые параметры страны по ее коду ID и языку strLang. | 3.2.2 | 14.10 |
| GetCityByID | Метод возвращает языконезависимые параметры города с кодом ID. | 3.2.2 | 14.10 |
| GetCityLangByID | Метод возвращает языкозависимые параметры города по его коду ID и языку strLang. | 3.2.2 | 14.10 |
| UpdateCountry | Метод изменяет параметры страны с кодом ID на новые параметры из массива arFields. | 3.2.2 | 14.10 |
| DeleteCountry | Метод удаляет страну с кодом ID. Связаные с этой страной местоположения не изменяются. | 3.2.2 | 14.10 |
| UpdateCity | Метод изменяет параметры города с кодом ID на значения из массива arFields. | 3.2.2 | 14.10 |
| DeleteCity | Метод удаляет город с кодом ID. Местоположение, с которым связан этот город, не изменяется. | 3.2.2 | 14.10 |
| UpdateLocation | Метод обновляет параметры местоположения с кодом ID в соответствии с параметрами из массива arFields. | 3.2.2 | 14.10 |
| Add | Метод добавляет новое местоположение включая страну и город местоположения, если нужно. | 3.2.2 | 14.10 |
| Update | Метод обновляет параметры местоположения с кодом ID в соответствии с параметрами из массива arFields. Обновляются также страна и город этого местоположения. | 3.2.2 | 14.10 |
| Delete | Метод удаляет местоположение с кодом ID. Метод также удаляет город этого местоположения, страну этого местоположения (если она не входит больше ни в одно другое местоположение), а так же связи этого местоположения с группами местоположений и службами доставки. | 3.2.2 | 14.10 |
| GetList | Метод возвращает набор местоположений, удовлетворяющих фильтру arFilter. | 3.3.0 | 14.10 |
| GetByID | Метод возвращает параметры местоположения с кодом ID, включая параметры страны и города. | 3.3.0 | 14.10 |
| GetCountryList | Метод возвращает набор стран по фильтру arFilter, отсортированный по массиву arOrder. | 3.3.0 | 14.10 |
| AddCountry | Метод добавляет новую страну с параметрами из массива arFields. | 3.3.0 | 14.10 |
| AddCity | Метод добавляет новый город с параметрами из массива arFields. | 3.3.0 | 14.10 |
| AddLocation | Метод добавляет новое местоположение на основании параметров массива arFields. | 3.3.0 | 14.10 |
| DeleteAll | Метод удаляет все местоположения из базы. | 3.3.5 | 14.10 |
| `GetByZIP($zipCode)` | Метод возвращает местоположение по почтовому индексу. | 3.3.5 | 14.10 |
| `GetLocationZIP($location)` | Метод возвращает почтовый индекс по местоположению. | 3.3.5 | 14.10 |

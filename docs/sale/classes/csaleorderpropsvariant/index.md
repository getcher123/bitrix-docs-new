# Класс CSaleOrderPropsVariant


#### Методы класса


| Метод | Описание | С версии |
| --- | --- | --- |
| GetList | Метод возвращает набор вариантов значений свойств заказа, удовлетворяющих фильтру. | 3.3.0 |
| GetByValue | Метод возвращает параметры варианта значения свойства заказа по коду свойства заказа и значению. | 3.3.0 |
| GetByID | Метод возвращает параметры варианта с кодом ID значения свойства заказа. | 3.3.0 |
| Update | Метод обновляет параметры варианта с кодом ID на значения из массива arFields. | 3.3.0 |
| Delete | Метод удаляет вариант с кодом ID значения свойства заказа. | 3.3.0 |
| DeleteAll | Метод удаляет все варианты значений для свойства с кодом ID заказа. | 3.3.0 |
| Add | Метод добавляет новый вариант для выбора значения в свойствах типа переключатель (RADIO) и списки (SELECT, MULTISELECT). | 3.3.5 |

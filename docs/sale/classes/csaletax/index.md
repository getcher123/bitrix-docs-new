# Класс CSaleTax


#### Методы класса


| Метод | Описание | С версии |
| --- | --- | --- |
| Update | Метод обновляет параметры налога с кодом ID. | 3.3.1 |
| Delete | Метод удаляет налог с кодом ID. | 3.3.1 |
| GetByID | Метод возвращает параметры налога с кодом ID. | 3.3.1 |
| GetList | Метод возвращает набор налогов, удовлетворяющих фильтру arFilter. | 3.3.1 |
| GetExemptList | Метод возвращает набор записей из таблицы освобождений от уплаты налогов, удовлетворяющих фильтру arFilter. | 3.3.1 |
| AddExempt | Метод добавляет новую запись в таблицу освобождения от налогов. | 3.3.1 |
| DeleteExempt | Метод удаляет записи, удовлетворяющие фильтру arFilter, из таблицы освобождений от уплаты налогов. | 3.3.1 |
| Add | Метод добавляет новый налог с параметрами из массива arFields. | 3.3.1 |

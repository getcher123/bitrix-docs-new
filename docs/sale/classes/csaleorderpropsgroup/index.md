# Класс CSaleOrderPropsGroup


#### Методы класса


| Метод | Описание | С версии |
| --- | --- | --- |
| GetList | Метод возвращает результат выборки записей из заказов в соответствии со своими параметрами. | 3.3.0 |
| GetByID | Метод возвращает параметры группы свойств заказа с кодом ID. | 3.3.0 |
| Update | Метод обновляет параметры группы заказов с кодом ID на параметры из массива arFields. | 3.3.0 |
| Delete | Метод удаляет группу свойств с кодом ID. | 3.3.0 |
| Add | Метод добавляет новую группу свойств заказа. Группа используется только для группировки свойств. | 3.3.5 |

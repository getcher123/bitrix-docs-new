# Интернет-магазин в ядре D7


### Список классов

Список классов модуля **Интернет-магазин** в ядре D7:


| Класс, пространство имен | Описание |
| --- | --- |
| **Archive**Класс для работы с архивацией заказов. | Класс для работы с архивацией заказов. |
| **Basket** | Класс для работы с корзиной. Наследник класса `\Bitrix\Sale\BasketItemCollection`. |
| **BasketBase** | Класс для работы с корзиной. |
| **BasketItemCollection** | Класс коллекции товарных позиций. |
| **BusinessValue** | Класс для работы с бизнес-смыслами. |
| **Configuration** | Класс для работы с настройками резервирования товаров, складского учета и настройками смены статусов интернет-магазина. |
| **Delivery** | Пространство имен, содержащее классы для работы со службами доставок. |
| **Discount** | Пространство имен, содержащее подпространства, классы и методы для расчета всех скидок (каталога и магазина) и округления цен для корзины или заказа. |
| **DiscountCouponsManager** | Класс для работы с купонами при расчетах. |
| **Fuser** | Класс для получения идентификатора покупателя. |
| **Internals** | Подпространство, содержащее классы для работы с таблицами модуля **Интернет-магазин**. |
| **Location** | Подпространство имен, содержащее классы для работы с местоположениями. |
| **Notify** | Класс для работы с уведомлениями. |
| **Order** | Класс для работы с заказами. |
| **OrderBase** | Базовый класс для работы с заказами. |
| **OrderDiscountManager** | Класс для работы с правилами корзины при расчетах. |
| **Payment** | Класс оплаты. |
| **PaymentCollection** | Класс коллекции оплат. |
| **PropertyValueCollectionBase** | Базовый класс коллекции свойств заказа. |
| **ProviderBase** | Базовый класс для работы с провайдерами магазина. |
| **Result** | Класс для работы с результатами. |
| **SalesZone** | Класс для задания зоны обслуживания магазином. |
| **Services** | Базовая общая часть служб доставок и платежных систем. |
| **Shipment** | Класс отгрузок. |
| **ShipmentCollection** | Класс коллекции отгрузок. |
| **ShipmentItem** | Класс элемента отгрузки. |
| **ShipmentItemStore** | Класс штрих-кодов элементов отгрузки. |
| **StatusBase** | Базовый класс статусов. |
| **Tax** | Класс налогов. |
| **TradingPlatform** | Класс для работы с торговыми платформами. |

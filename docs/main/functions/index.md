# Функции главного модуля

При разработке сайта на базе системы "1С-Битрикс: Управление сайтом" можно применять функции следующих групп:


- Строки - функции для работы со строками и текстом. Массивы- функции для работы с массивами. Даты и время- функции для работы с датами, временем и календарем. Файлы и каталоги- функции для работы с файловой системой сайта. Элементы HTML- функции для автоматического построения элементов форм. Фильтр и сортировка- функции для построения и работы с фильтрами и сортировками. Модули- функции для работы с модульной системой сайта. Языковые файлы- функции для локализации скриптов сайта (для показа сайта на разных языках). Отладка- функции для облегчения отладки сайта. Прочее- прочие функции.

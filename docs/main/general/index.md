# Введение

Справочная информация по технологиям и принципам заложенным в систему. Кроме этого рекомендуется к изучению: - **Список терминов** - определения основных терминов используемых в системе; **Структура файлов**- описание структуры файлов системы **Права доступа**- описание идеологии распределения прав доступа Специальные константы- специальные константы используемые в системе Специальные переменные- специальные переменные используемые в системе Работа с датой и временем- примеры основных операций связанных в датой и временем **Навигационная цепочка**- описание принципов работы с навигационной цепочкой **Меню**- описание принципов работы с меню Технологии- описание основных технологий реализованных в системе **Модули**- описание принципов модульности и основных вопросов связанных с разработкой собственного модуля **Сайты**- принципы многосайтовости Языки- перевод скриптов системы на другие языки **Страницы**- структура страницы **Компоненты**- описание принципов работы с компонентами

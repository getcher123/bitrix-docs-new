# Mail


```
bool
CFormResult::Mail(
	int result_id,
	mixed template_id = false
)
```

Создает почтовое событие для отсылки данных [результата](../../terms.md#result) по e-mail. Возвращает "true" в случае успеха, в противном случае - "false". Метод нестатический.


#### Параметры метода


| Параметр | Описание |
| --- | --- |
| *result_id* | ID [результата](../../terms.md#result). |
| *template_id* | ID почтового шаблона. Параметр необязательный. По умолчанию - "false" (будут использованы почтовые шаблоны из настроек соответствующей веб-формы). |


#### Примеры использования


```
<?
$RESULT_ID = 189; // ID результата

// создадим почтовое событие для отсылки по EMail данных результата
if (CFormResult::Mail($RESULT_ID))
{
	echo "Почтовое событие успешно создано.";
}
else // ошибка
{
	global $strError;
	echo $strError;
}
?>
```

# Delete


### Описание и параметры


```
bool
CFormResult::Delete(
	int result_id,
	string check_rights = "Y"
)
```

Удаляет указанный [результат](../../terms.md#result). В случае успеха метод возвращает "true", иначе - "false". Метод нестатический.


#### Параметры метода


| Параметр | Описание |
| --- | --- |
| *result_id* | ID [результата](../../terms.md#result). |
| *check_rights* Флаг необходимости проверки прав текущего пользователя. Возможны следующие значения: - **Y** - права необходимо проверить; **N**- права не нужно проверять. Для успешного удаления [результата](../../terms.md#result)необходимо обладать следующими [правами](../../permissions.md): 1. На веб-форму, к которой принадлежит редактируемый результат: **[20] Работа со всеми результатами в соответствии с их статусами** или, в случае, если вы являетесь создателем удаляемого результата, достаточно права **[15] Работа со своим результатом в соответствии с его статусом.** На статус в котором находится редактируемый результат необходимо иметь право: **[DELETE] удаление.** Параметр необязательный. По умолчанию - "Y" (права необходимо проверить). | Флаг необходимости проверки прав текущего пользователя. Возможны следующие значения: - **Y** - права необходимо проверить; **N**- права не нужно проверять. Для успешного удаления [результата](../../terms.md#result) необходимо обладать следующими [правами](../../permissions.md): 1. На веб-форму, к которой принадлежит редактируемый результат: **[20] Работа со всеми результатами в соответствии с их статусами** или, в случае, если вы являетесь создателем удаляемого результата, достаточно права **[15] Работа со своим результатом в соответствии с его статусом.** На статус в котором находится редактируемый результат необходимо иметь право: **[DELETE] удаление.** Параметр необязательный. По умолчанию - "Y" (права необходимо проверить). |

### Примеры использования


```
<?
$RESULT_ID = 189; // ID результата

// удалим результат с проверкой прав текущего пользователя
if (CFormResult::Delete($RESULT_ID))
{
	echo "Результат # ".$RESULT_ID." успешно удален.";
}
else // ошибка
{
	global $strError;
	echo $strError;
}
?>
```

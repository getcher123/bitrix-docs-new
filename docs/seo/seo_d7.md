# Поисковая оптимизация в ядре D7


#### Классы

Список классов модуля **Поисковая оптимизация** в ядре D7


| Класс, простраство имён | Описание |
| --- | --- |
| **Adv** | Пространство имён классов для работы с рекламными кампаниями, объявлениями, баннерами. |
| **AdvEntity** | Базовый класс для сущностей внешнего сервиса рекламы. |
| **Engine** | Пространство имён транспортных классов для работы с Яндекс.Директ и класса для поддержки централизованной авторизации при работе с Яндекс.Директ. |
| **SitemapFile** | Класс для работы с файлами sitemap. |
| **SitemapIblock** | Класс, отвечающий за обслуживание событий инфоблоков с целью автоматического отражения изменения в файлах карты сайта. |
| **SitemapIblockTable** | Класс для работы с хранилищем привязок инфоблоков к механизму генерации файлов карты сайта. |
| **SitemapRuntime** | Класс для работы с временным файлом карты сайта. |
| **SitemapTable** | Класс для работы с хранилищем настроек генерации файлов карты сайта. |

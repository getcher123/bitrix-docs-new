# Социальная сеть в ядре D7

Список классов модуля **Социальная сеть**в ядре D7.
| Класс, пространство имен | Описание |
| --- | --- |
| **Item**Пространство имён для работы с сущностями на более высоком уровне, чем непосредственно с записями в таблице. | Пространство имён для работы с сущностями на более высоком уровне, чем непосредственно с записями в таблице. |
| **Ui**Пространство имён пользовательских интерфейсов. | Пространство имён пользовательских интерфейсов. |
| **UserToGroupTable**Класс API низкого уровня для работы с отношениями пользователь-группа (роль пользователя в группе). | Класс API низкого уровня для работы с отношениями пользователь-группа (роль пользователя в группе). |
| **Util**Класс вспомогательных методов, служащих различным целям в рамках реализации логики социальной сети. | Класс вспомогательных методов, служащих различным целям в рамках реализации логики социальной сети. |
| **WorkgroupFavoritesTable**Класс для методов работы с избранными рабочими группами. | Класс для методов работы с избранными рабочими группами. |

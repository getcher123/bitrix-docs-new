# Классы модуля подписки и рассылки

Следующие PHP классы используются в модуле подписки и рассылки.


#### Классы


| Класс | Описание | С версии |
| --- | --- | --- |
| [CPosting](cposting/index.md) | Класс для работы с выпусками новостей подписки. | 3.3.0 |
| [CRubric](crubric/index.md) | Класс для работы с рубриками подписки. | 3.1.1 |
| [CSubscription](csubscriptiongeneral/index.md) | Класс для работы с подписками на рассылки. | 3.1.1 |
| [CPostingTemplate](cpostingtemplate/index.md) | Класс для работы с шаблонами. | 4.0.5 |

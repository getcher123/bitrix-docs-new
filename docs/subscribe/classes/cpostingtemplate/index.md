# Класс CPostingTemplate

**CPostingTemplate** - класс для работы с шаблонами генерации выпусков подписки.


#### Методы класса


| Метод | Описание | C версии |
| --- | --- | --- |
| GetList | Метод возвращает список шаблонов. | 4.0.5 |
| GetByID | Метод возвращает шаблон по его идентификатору (относительному пути). | 4.0.5 |
| IsExists | Метод проверяет существование каталога шаблона. | 4.0.5 |
| Execute | Метод выбирает шаблон для генерации выпуска рассылки в соответствии с расписанием. | 4.0.5 |
| AddPosting | Метод генерации выпуска на основании шаблона. | 4.0.5 |

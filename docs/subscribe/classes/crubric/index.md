# Класс CRubric

**CRubric** - класс для работы с рубриками подписки.


#### Методы класса


| Метод | Описание | С версии |
| --- | --- | --- |
| Add | Метод добавляет рассылку. | 5.0.2 |
| Delete | Метод удаляет рассылку. | 5.0.2 |
| GetByID | Метод выбирает одну рассылку по ее идентификатору. | 5.0.2 |
| GetList | Метод возвращает список рассылок по фильтру. | 5.0.2 |
| GetSubscriptionCount | Метод возвращает количество подписчиков на указанную рассылку. Учитываются как подтвержденные, так и неподтвержденные подписчики. | 5.0.2 |
| Update | Метод модифицирует рассылку. | 5.0.2 |

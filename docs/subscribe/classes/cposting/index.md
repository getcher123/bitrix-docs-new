# Класс CPosting

**CPosting** - класс для работы с выпусками новостей подписки.


#### Методы класса


| Метод | Описание | С версии |
| --- | --- | --- |
| GetList | Метод возвращает список выпусков по фильтру. | 3.1.1 |
| Lock | Метод возвращает true при успешной блокировке выпуска и false при неуспешной. | 4.0.5 |
| UnLock | Метод возвращает true при успешном снятии блокировки выпуска и false при неуспешном. | 4.0.7 |

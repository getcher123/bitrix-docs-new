# Класс CPostingGeneral

**CPostingGeneral** - класс для работы с выпусками новостей подписки.


#### Методы класса


| Метод | Описание | С версии |
| --- | --- | --- |
| Add | Метод добавляет выпуск. | 3.1.1 |
| Delete | Метод удаляет выпуск по его идентификатору. | 3.1.1 |
| GetByID | Метод возвращает выпуск по его идентификатору. | 3.1.1 |
| GetEmails | Метод возвращает массив адресов, по которым выпуск будет отправлен. | 3.3.2 |
| GetGroupList | Метод возвращает выборку групп пользователей, на которые будет отправлен выпуск. | 3.1.1 |
| UpdateGroups | Метод изменяет информацию о привязке выпуска к группам пользователей. | 3.1.1 |
| GetRubricList | Метод возвращает выборку рассылок, на которые будет отправлен выпуск. | 3.1.1 |
| UpdateRubrics | Метод изменяет информацию о привязке выпуска к рубрикам подписки. | 3.1.1 |
| SendMessage | Метод возвращает true при успешной отправке, false при неуспешной, "CONTINUE" при частичной отправке. При неуспешной отправке переменная LAST_ERROR класса содержит сообщение об ошибке. | 3.2.0 |
| Update | Метод изменяет информацию о выпуске по его идентификатору. | 3.1.1 |
| SaveFile | Метод добавляет вложение к выпуску сохраняя и регистрируя его в таблице файлов (b_file). | 4.0.5 |
| DeleteFile | Метод удаляет одно или все вложения выпуска. | 4.0.5 |
| GetFileList | Метод возвращает выборку вложений выпуска. | 4.0.5 |
| AutoSend | Метод для отправки выпуска с помощью cron'а или агента. | 4.0.5 |
| ChangeStatus | Метод изменяет статус выпуска. | 4.0.5 |
| GetEmailStatuses | Метод возвращает массив статусов в очереди выпуска на отправку. | 8.5.3 |
| GetEmailsByStatus | Метод возвращает выборку из очереди на отправку. | 8.5.3 |

# Классы модуля обучения

Следующие PHP классы используются в модуле обучения.


#### Классы


| Класс | Описание | С версии | До версии |
| --- | --- | --- | --- |
| CCourse | Класс для работы с курсами. | 5.0.3 |  |
| CChapter | Класс для работы с главами. | 5.1.0 | актуален до 12.0.0 |
| CLesson | Класс для работы с уроками. | 5.1.0 | актуален до 12.0.0 |
| CLQuestion | Класс для работы с вопросами. | 5.0.3 |  |
| CLAnswer | Класс для работы с ответами. | 5.0.6 |  |
| CTest | Класс для работы с тестами. | 5.1.0 |  |
| CTestAttempt | Класс для работы с попытками прохождения теста. | 5.1.0 |  |
| CTestResult | Класс для работы с планом тестирования. | 5.1.0 |  |
| CGradeBook | Класс для работы с журналом. | 5.1.0 |  |
| CCertification | Класс для работы с сертификатами. | 5.1.0 |  |
| CStudent | Класс для работы c профилем студента. | 5.1.0 |  |
| CCoursePackage | Класс для экспорта курса. | 5.0.3 |  |
| CCourseImport | Класс для импорта курса. | 5.0.3 |  |

# learning — Обучение

Перед использованием модуля **Обучение** необходимо проверить, установлен ли он, и подключить его при помощи конструкции:


```
<?
if (CModule::IncludeModule("learning"))
{
	//здесь можно использовать функции модуля
}
?>
```


Модуль включает в себя следующие Компоненты 2.0:

| **Компоненты модуля** |  |  |
| --- | --- | --- |
| **Название** | **Имя в пространстве имен bitrix** | **Описание** |
| **Детальный вывод главы** | **learning.chapter.detail** | Производит детальный вывод главы курса. |
| **Учебный курс (комплексный компонент)** | **learning.course** | Осуществляет полноценный вывод учебного курса. |
| **Все материалы курса** | **learning.course.contents** | Выводит все материалы курса на одной странице. |
| **Детальный вывод курса** | **learning.course.detail** | С помощью компонента производится детальный вывод курса. |
| **Список курсов** | **learning.course.list** | Выводит список курсов. |
| **Дерево курса** | **learning.course.tree** | Выводит иерархию глав и уроков курса. |
| **Детальный вывод урока** | **learning.lesson.detail** | Производит детальный вывод урока курса. |
| **Отчет по курсам** | **learning.student.certificates** | Выводит список пройденных и непройденных курсов текущего пользователя. |
| **Журнал студента** | **learning.student.gradebook** | Отображает результаты прохождения тестов. |
| **Профиль студента** | **learning.student.profile** | Отображает результаты прохождения тестов. |
| **Резюме студента** | **learning.student.transcript** | Отображает информацию о студенте и список полученных им сертификатов. |
| **Контрольный тест** | **learning.test** | Отображает контрольный тест курса. |
| **Список тестов** | **learning.test.list** | Выводит список активных тестов курса. |
| **Самопроверка** | **learning.test.self** | Выводит тест для самопроверки. |

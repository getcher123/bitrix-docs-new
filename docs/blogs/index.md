# blogs — Блоги

Перед использованием модуля необходимо проверить, установлен ли он, и подключить его при помощи конструкции:


```
<?
if (CModule::IncludeModule("blog"))
{
	//здесь можно использовать функции модуля
}
?>
```


Модуль включает в себя следующие Компоненты 2.0:

| **Компоненты модуля** |  |  |
| --- | --- | --- |
| **Название** | **Имя в пространстве имен bitrix** | **Описание** |
| **Блоги (комплексный компонент)** | **blog** | Создает полноценный публичный интерфейс блога. |
| **Сообщения блога** | **blog.blog** | Выводит сообщения блога с возможностью их фильтрации по категории и дате. |
| **Черновики сообщений блога** | **blog.blog.draft** | Выводит неопубликованные сообщения текущего пользователя блога. |
| **Редактирование блога** | **blog.blog.edit** | Служит для изменения параметров и настройки блога. |
| **Лучшие сообщения блога** | **blog.blog.favorite** | Выводит список лучших сообщения блога на основе отметок владельца блога (при создании сообщений владелец блога указывает индекс сортировки сообщения). |
| **Календарь** | **blog.calendar** | Выводит календарь сообщений блога. |
| **Теги блога** | **blog.category** | Служит для редактирования тегов сообщений блога. |
| **Комментируемые сообщения** | **blog.commented_posts** | Выводит наиболее комментируемые сообщения блогов за заданный период. |
| **Лента друзей** | **blog.friends** | Выводит сообщения друзей пользователя блога. |
| **Блоги группы** | **blog.group.blog** | Выводит блоги группы. |
| **Группы блогов** | **blog.groups** | Выводит непустые группы блогов. |
| **Информация о блоге** | **blog.info** | Выводит информацию о блоге со списком тегов. |
| **Меню блогов** | **blog.menu** | Выводит меню блога. |
| **Новые блоги** | **blog.new_blogs** | Выводит новые блоги. |
| **Новые комментарии** | **blog.new_comments** | Выводит последние комментарии к сообщениям блогов. |
| **Новые сообщения** | **blog.new_posts** | Выводит последние сообщения блогов текущего сайта. |
| **Последние сообщения всех блогов** | **blog.new_posts.list** | Выводит последние сообщения блогов с постраничной навигацией. |
| **Популярные блоги** | **blog.popular_blogs** | Выводит популярные блоги. |
| **Популярные сообщения** | **blog.popular_posts** | Выводит наиболее просматриваемые сообщения блогов за заданный период. |
| **Сообщение детально** | **blog.post** | Выводит сообщение блога подробно. |
| **Комментарии к сообщению** | **blog.post.comment** | Выводит комментарии к сообщению блога с возможностью добавления нового комментария. |
| **Редактирование сообщения** | **blog.post.edit** | Служит для создания новых или редактирования существующих сообщения блога. |
| **Trackback'и сообщения** | **blog.post.trackback** | Выводит Trackback-адрес и Trackback'и к сообщению блога. |
| **Получение Trackback'ов к сообщению** | **blog.post.trackback.get** | Получает Trackback к сообщению блога. |
| **RSS блога** | **blog.rss** | Выводит RSS блогов в заданном формате. |
| **RSS блогов по группам/сайтам** | **blog.rss.all** | Выводит RSS блогов группы или сайта в заданном формате. |
| **Ссылки на RSS блога** | **blog.rss.link** | Выводит ссылки на RSS блога. |
| **Поиск по блогам** | **blog.search** | Выводит непустые группы блогов. |
| **Профиль пользователя** | **blog.user** | Позволяет просмотреть и изменить профиль пользователя. |
| **Группы пользователей блога** | **blog.user.group** | Служит для редактирования группы пользователей блога. |
| **Настройка пользователей блога** | **blog.user.settings** | Позволяет настраивать пользователей блога: добавлять их в друзья, удалять. |
| **Настройка прав пользователя блога** | **blog.user.settings.edit** | Позволяет настраивать привязку пользователя к группам блога. |
| **Получение данных в формате metaWeblog** | **blog.metaweblog** | Осуществляет импорт в формате MetaWeblog. |
| **Неопубликованные сообщения блога** | **blog.blog.moderation** | Выводит сообщения блога, требующие модерации. |

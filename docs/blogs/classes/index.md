# Классы

Следующие PHP классы используются в модуле блогов.


#### Классы


| Класс | Описание | С версии |
| --- | --- | --- |
| CBlog | Класс для работы с блогами. | 5.0.1 |
| CBlogCandidate | Класс для работы с желающими стать друзьями. | 7.1.2 |
| CBlogCategory | Класс для работы с категориями сообщений блога. | 6.5.1 |
| CBlogComment | Класс для работы с комментариями к сообщениям. | 5.9.1 |
| CBlogGroup | Класс для работы с группами блогов. | 7.1.2 |
| CBlogImage | Класс для работы с изображениями, прикреплёнными к сообщениям в блоге. | 8.5.3 |
| CBlogPost | Класс для работы с сообщениями блогов. | 5.0.2 |
| CBlogSitePath | Класс для работы с путями к публичной части блогов. | 5.9.1 |
| CBlogTrackBack | Класс для работы с TrackBack'ами к сообщениям. | 5.9.0 |
| CBlogUser | Класс для работы с пользователями блогов. | 5.0.2 |
| CBlogUserGroup | Класс для работы с группами пользователей блогов. | 7.1.2 |
| blogTextParser | Класс для форматирования сообщений. | 5.0.2 |

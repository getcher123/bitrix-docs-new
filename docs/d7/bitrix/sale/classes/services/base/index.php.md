# Base

Источник: https://dev.1c-bitrix.ru/api_d7/bitrix/sale/classes/services/base/index.php

**Base** – базовые сущности, от которых наследуются некоторые сущности служб доставок и платежных систем.

| Класс | Описание | С версии |
| --- | --- | --- |
| [Restriction](restriction/index.php.md) | Базовая сущность для ограничений. |  |
| [RestrictionInfo](restrictioninfo/index.php.md) | Класс, описывающий формат, в котором ограничения хранятся в коллекции. |  |
| [RestrictionManager](restrictionmanager/index.php.md) | Менеджер ограничений для сервисов. |  |

# Интерфейс редактирования сущности

Источник: https://dev.1c-bitrix.ru/api_d7/bitrix/ui/entity_editor/index.php

Стандартный интерфейс для редактирования сущности. Используется на странице пользователя, в карточке товара, в документах складского учёта, в карточках большинства сущностей CRM (через наследников в модуле CRM).

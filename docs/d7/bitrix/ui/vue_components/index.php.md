# Vue-компоненты

Источник: https://dev.1c-bitrix.ru/api_d7/bitrix/ui/vue_components/index.php

Vue-компоненты — это готовые элементы интерфейса, которые адаптированы для использования в Vue 3. Они позволяют встраивать такие компоненты, как меню, кнопки и попапы, в Vue-приложения.

| Компонент | Описание |
| --- | --- |
| [ui.system.menu](system_menu.php.md) | Выпадающее меню, которое предназначено для отображения списка действий: настройки, дополнительные опции или контекстные команды. |

# Компонент-обёртка для слайдера

Источник: https://dev.1c-bitrix.ru/api_d7/bitrix/ui/sidepanel_wrapper/index.php

Если в iframe'е [слайдера](../../../../js_lib/sidepanel/index.md) открыть обычную страницу сайта, то она покажется вместе с общим дизайном (меню, шапка, подвал). Для этого режима необходимо адаптировать вывод компонента.

Компонент **bitrix:ui.sidepanel.wrapper** решает эту задачу. Он является оберткой для вывода целевого компонента в слайдере.

Компонент доступен с версии 17.6.5 модуля **UI**.

- [Начало работы](https://dev.1c-bitrix.ru/api_d7/bitrix/ui/sidepanel_wrapper/start.php)
- [Параметры поведения](https://dev.1c-bitrix.ru/api_d7/bitrix/ui/sidepanel_wrapper/action_params.php)
- [Параметры отображения](https://dev.1c-bitrix.ru/api_d7/bitrix/ui/sidepanel_wrapper/view_params.php)

# Json

Источник: https://dev.1c-bitrix.ru/api_d7/bitrix/main/web/json/index.php

**Json** - класс реализует работу с текстовым форматом обмена данными JavaScript Object Notation (JSON).

| Метод | Описание | С версии |
| --- | --- | --- |
| [decode](https://dev.1c-bitrix.ru/api_d7/bitrix/main/web/json/decode.php) | Метод выполняет декодирование строки JSON и конвертирует данные в переменные PHP. |  |
| [encode](https://dev.1c-bitrix.ru/api_d7/bitrix/main/web/json/encode.php) | Метод возвращает строку, содержащую JSON представление переменной $data. |  |

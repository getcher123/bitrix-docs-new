# HttpHeaders

Источник: https://dev.1c-bitrix.ru/api_d7/bitrix/main/web/httpheaders/index.php

**HttpHeaders** — класс для работы с заголовками.

| Метод | Описание |
| --- | --- |
| [add](https://dev.1c-bitrix.ru/api_d7/bitrix/main/web/httpheaders/add.php) | Метод добавляет заголовок |
| [clear](https://dev.1c-bitrix.ru/api_d7/bitrix/main/web/httpheaders/clear.php) | Метод очищает все заголовки |
| [delete](https://dev.1c-bitrix.ru/api_d7/bitrix/main/web/httpheaders/delete.php) | Метод удаляет заголовок по имени |
| [get](https://dev.1c-bitrix.ru/api_d7/bitrix/main/web/httpheaders/get.php) | Метод возвращает заголовок по его имени |
| [getCharset](https://dev.1c-bitrix.ru/api_d7/bitrix/main/web/httpheaders/getcharset.php) | Метод возвращает кодировку из Content-Type заголовка |
| [getContentDisposition](https://dev.1c-bitrix.ru/api_d7/bitrix/main/web/httpheaders/getcontentdisposition.php) | Метод возвращает disposition-type из Content-Disposition заголовка |
| [getContentType](https://dev.1c-bitrix.ru/api_d7/bitrix/main/web/httpheaders/getcontenttype.php) | Метод возвращает тип контента из Content-Type заголовка |
| [getFilename](https://dev.1c-bitrix.ru/api_d7/bitrix/main/web/httpheaders/getfilename.php) | Метод возвращает имя файла из Content-disposition заголовка |
| [has](https://dev.1c-bitrix.ru/api_d7/bitrix/main/web/httpheaders/has.php) | Метод возвращает `true`, если заголовок с указанным именем существует |
| [set](https://dev.1c-bitrix.ru/api_d7/bitrix/main/web/httpheaders/set.php) | Метод устанавливает значение заголовка |
| [toArray](https://dev.1c-bitrix.ru/api_d7/bitrix/main/web/httpheaders/toarray.php) | Метод возвращает заголовки в виде исходного массива |
| [toString](https://dev.1c-bitrix.ru/api_d7/bitrix/main/web/httpheaders/tostring.php) | Метод возвращает строковое представление запроса HTTP |

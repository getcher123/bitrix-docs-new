# setVersion

Источник: https://dev.1c-bitrix.ru/api_d7/bitrix/main/web/httpclient/setversion.php

```
void public
\Bitrix\Main\Web\HttpClient::setVersion(
	string $value
);
```

Нестатический метод устанавливает версию HTTP протокола. В версии 1.1 возможен фрагментированный ответ.

#### Параметры

| Параметр | Описание | Версия |
| --- | --- | --- |
| $value | Версия "1.0" или "1.1" (По умолчанию "1.0"). |  |

# getResult

Источник: https://dev.1c-bitrix.ru/api_d7/bitrix/main/web/httpclient/getresult.php

```
string public
\Bitrix\Main\Web\HttpClient::getResult(
);
```

Нестатический метод возвращает строку сущности HTTP ответа.

> **Важно**! если установлен [OutputStream](setoutputstream.php.md), то результатом будет пустая строка.

Без параметров

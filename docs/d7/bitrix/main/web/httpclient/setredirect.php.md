# setRedirect

Источник: https://dev.1c-bitrix.ru/api_d7/bitrix/main/web/httpclient/setredirect.php

```
void public
\Bitrix\Main\Web\HttpClient::setRedirect(
	boolean $value,
	null|integer $max = null
);
```

Нестатический метод устанавливает опции редиректа.

#### Параметры

| Параметр | Описание | Версия |
| --- | --- | --- |
| $value | Если *true*, выполняется редирект (по умолчанию *true*). |  |
| $max | Максимально возможное число редиректов. |  |

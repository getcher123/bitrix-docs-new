# setStreamTimeout

Источник: https://dev.1c-bitrix.ru/api_d7/bitrix/main/web/httpclient/setstreamtimeout.php

```
void public
\Bitrix\Main\Web\HttpClient::setStreamTimeout(
	integer $value
);
```

Нестатический метод устанавливает таймаут чтения потока сокетов.

#### Параметры

| Параметр | Описание | Версия |
| --- | --- | --- |
| $value | Таймаут чтения потока  в секундах; "0" - нет ограничений (по умолчанию 60). |  |

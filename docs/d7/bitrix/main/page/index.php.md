# Page

Источник: https://dev.1c-bitrix.ru/api_d7/bitrix/main/page/index.php

**Page** - пространство имён для классов, работающих со страницей продукта.

| Класс | Описание | С версии |
| --- | --- | --- |
| [Frame](https://dev.1c-bitrix.ru/api_d7/bitrix/main/page/frame/index.php) | Класс записывает все содержимое страницы в кеш, предварительно вырезав из нее контент динамических областей. | 14.0.1 |
| [Asset](https://dev.1c-bitrix.ru/api_d7/bitrix/main/page/asset/index.php) | Класс отвечает за подключение стилей и скриптов. | 14.9.4 |
| [FrameBuffered](https://dev.1c-bitrix.ru/api_d7/bitrix/main/page/framebuffered/index.php) | Класс для создания буферизированных динамических областей. | 14.9.4 |
| [FrameLocker](https://dev.1c-bitrix.ru/api_d7/bitrix/main/page/framelocker/index.php) | Класс для блокирования буферизированных динамических областей. | 16.0.12 |

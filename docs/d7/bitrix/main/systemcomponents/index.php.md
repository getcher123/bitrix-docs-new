# Системные компоненты

Источник: https://dev.1c-bitrix.ru/api_d7/bitrix/main/systemcomponents/index.php

В разделе содержится информация о работе с системными компонентами.

**Внимание!** Компоненты являются **системными** и в визуальном редакторе не отображаются. В физической структуре сайта их можно найти в папке `/bitrix/components/bitrix/[название_компонента]`

| Компонент | Описание |
| --- | --- |
| [main.ui.grid и main.ui.filter](gridandfilter/index.php.md) | **main.ui.grid** выводит информацию в табличной форме. **main.ui.filter** выводит фильтр + поиск. Компоненты могут работать в связке, что позволяет вывести информацию в виде таблицы с возможностью фильтрации и поиска. |
| [main.user.selector](mainuserselector/index.php.md) | Компонент представляет собой готовый к встраиванию в форму контрол для выбора сущностей различных типов. |
| [main.field.*](mainfield.php.md) | Компоненты отрисовки пользовательских полей. |

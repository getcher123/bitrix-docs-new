# CustomConnectors

Источник: https://dev.1c-bitrix.ru/api_d7/bitrix/imconnector/customconnectors/index.php

**CustomConnectors** - класс методов для подключения сторонних коннекторов.

| Метод | Описание | С версии |
| --- | --- | --- |
| [sendMessages](sendmessages.php.md) | Отправка сообщений |  |
| [updateMessages](updatemessages.php.md) | Обновление сообщений |  |
| [sendStatusDelivery](sendstatusdelivery.php.md) | Подтверждение доставки сообщения от Открытой Линии во внешнюю систему. |  |
| [deleteMessages](deletemessages.php.md) | Удаление сообщений в ОЛ. |  |
| [deactivateConnectors](deactivateconnectors.php.md) | Отключение настроенного коннектора |  |

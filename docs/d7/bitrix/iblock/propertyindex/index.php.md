# PropertyIndex

Источник: https://dev.1c-bitrix.ru/api_d7/bitrix/iblock/propertyindex/index.php

Пространство имен `\Bitrix\Iblock\PropertyIndex` содержит классы для работы с индексами инфоблоков.

| Класс | Описание |
| --- | --- |
| [Dictionary](https://dev.1c-bitrix.ru/api_d7/bitrix/iblock/propertyindex/dictionary/index.php)Класс для работы со словарем строковых значений свойств. |  |
| [Element](https://dev.1c-bitrix.ru/api_d7/bitrix/iblock/propertyindex/element/index.php)Абстрактный класс для работы с наследуемыми свойствами в элементах инфоблоков. |  |
| [Facet](https://dev.1c-bitrix.ru/api_d7/bitrix/iblock/propertyindex/facet/index.php)Класс для построения запросов с использованием фасетных индексов. |  |
| [Indexer](https://dev.1c-bitrix.ru/api_d7/bitrix/iblock/propertyindex/indexer/index.php)Класс отвечает за процесс индексации. |  |
| [Manager](https://dev.1c-bitrix.ru/api_d7/bitrix/iblock/propertyindex/manager/index.php)Класс для управления индексами. |  |
| [QueryBuilder](https://dev.1c-bitrix.ru/api_d7/bitrix/iblock/propertyindex/querybuilder/index.php)Класс для работы с запросами. |  |
| [Storage](https://dev.1c-bitrix.ru/api_d7/bitrix/iblock/propertyindex/storage/index.php)Класс для работы с таблицей индексов инфоблока. |  |

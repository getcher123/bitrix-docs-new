# Template

Источник: https://dev.1c-bitrix.ru/api_d7/bitrix/iblock/template/index.php

Пространство имен `\Bitrix\Iblock\Template` содержит классы для работы с шаблонами SEO свойств.

| Класс, пространство имен | Описание |
| --- | --- |
| [Engine](https://dev.1c-bitrix.ru/api_d7/bitrix/iblock/template/engine/index.php)Класс для запуска обработки шаблонов SEO свойств. |  |
| [Entity](https://dev.1c-bitrix.ru/api_d7/bitrix/iblock/template/entity/index.php)Пространство содержит классы для работы с сущностями в шаблонах SEO свойств. |  |
| [Functions](https://dev.1c-bitrix.ru/api_d7/bitrix/iblock/template/functions/index.php)Пространство содержит классы для работы с функциями шаблонов SEO свойств. |  |
| [Helper](https://dev.1c-bitrix.ru/api_d7/bitrix/iblock/template/helper/index.php)Вспомогательный класс для работы с шаблонами SEO свойств. |  |
| [NodeBase](https://dev.1c-bitrix.ru/api_d7/bitrix/iblock/template/nodebase/index.php)Базовый класс, отвечающий за логику преобразования данных в значения наследуемых свойств. |  |
| [NodeEntityField](https://dev.1c-bitrix.ru/api_d7/bitrix/iblock/template/nodeentityfield/index.php)Класс отвечает за логику преобразования данных полей сущности в значения наследуемых свойств. |  |
| [NodeFunction](https://dev.1c-bitrix.ru/api_d7/bitrix/iblock/template/nodefunction/index.php)Класс отвечает за логику работы функций наследуемых свойств. |  |
| [NodeRoot](https://dev.1c-bitrix.ru/api_d7/bitrix/iblock/template/noderoot/index.php)Класс отвечает за логику преобразования данных дочерних сущностей. |  |
| [NodeText](https://dev.1c-bitrix.ru/api_d7/bitrix/iblock/template/nodetext/index.php)Класс отвечает за логику работы с текстом в наследуемых свойствах. |  |

# Список классов


| Класс | Описание | С версии |

| CAdv | Класс для работы с рекламными кампаниями. | создан ранее 8.6.3 |
| CAutoDetect | Класс для поиска неизвестных UserAgent'ов. | создан ранее 8.6.3 |
| CCountry | Класс для получения данных по трафику в разрезе по странами. | создан ранее 8.6.3 |
| CGuest | Класс для получения данных по посетителям сайта. | создан ранее 8.6.3 |
| CHit | Класс для получения данных по хитами посетителей. | создан ранее 8.6.3 |
| CPage | Класс для получения данных о посещенных страницах сайта. | создан ранее 8.6.3 |
| CPath | Класс для получения данных о путях по сайту. | создан ранее 8.6.3 |
| CPhrase | Класс для получения данных по поисковым фразам. | создан ранее 8.6.3 |
| CReferer | Класс для получения данных о ссылающихся сайтах. | создан ранее 8.6.3 |
| CSearcher | Класс для работы с поисковыми системами. | создан ранее 8.6.3 |
| CSearcherHit | Класс для получения данных о хитах поисковых систем. | создан ранее 8.6.3 |
| CSession | Класс для получения данных о сессиях посетителей. | создан ранее 8.6.3 |
| CStatEvent | Класс для работы с событиями. | создан ранее 8.6.3 |
| CStatEventType | Класс для работы с типами событий. | создан ранее 8.6.3 |
| CStatistics | Класс содержащий общие методы работы с модулем "Статистика". | создан ранее 8.6.3 |
| CStopList | Класс для работы со стоп-листом. | создан ранее 8.6.3 |
| CTraffic | Класс для получения общих данных по посещаемости сайта. | создан ранее 8.6.3 |
| CUserOnline | Класс для получения данных о недавних посетителях сайта и их сессиях. | создан ранее 8.6.3 |

# Переменные окружения

Модуль статистики реализует следующие сессионные переменные:
| **$_SESSION["SESS_SESSION_ID"]** | ID сессии. |
| --- | --- |
| **$_SESSION["SESS_GUEST_ID"]** | ID посетителя. |
| **$_SESSION["SESS_GUEST_NEW"]** | "Y" - посетитель впервые на сайте; "N" - посетитель вернулся на сайт. |
| **$_SESSION["SESS_COUNTRY_ID"]** | двухсимвольный идентификатор страны посетителя. |
| **$_SESSION["SESS_ADV_ID"]** | ID рекламной кампании по которой посетитель делает прямой заход. |
| **$_SESSION["SESS_LAST_ADV_ID"]** | ID последней рекламной кампании посетителя по которой он сделал прямой заход, либо по которой вернулся. |
| **$_SESSION["SESS_LAST_USER_ID"]** | ID пользователя после его последней авторизации. |
| **$_SESSION["SESS_LAST_USER_AUTH"]** | "Y" - в последний раз посетитель был авторизован; "N" - нет. |
| **$_SESSION["SESS_GUEST_FAVORITES"]** | "Y" - посетитель уже добавлял сайт в избранное; "N" - нет. |
| **$_SESSION["FROM_SEARCHER_ID"]** | ID поисковика с которого посетитель пришел на сайт. |
| **$_SESSION["SESS_SEARCH_PHRASE"]** | поисковая фраза с которой посетитель пришел на сайт. |
| **$_SESSION["SESS_SEARCHER_ID"]** | ID поисковика робот которого индексирует сайт. |

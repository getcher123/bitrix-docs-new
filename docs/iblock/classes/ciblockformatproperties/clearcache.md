# clearCache


```
CIBlockFormatProperties::clearCache()
```

Метод очищает кеш в памяти данных класса, заполняемый в методе [GetDisplayValue](getdisplayvalue.md). Рекомендуется вызывать в конце работы компонент, использующих метод [GetDisplayValue](getdisplayvalue.md). Метод статический.


#### Параметры вызова

Без параметров.

# _CIBElement

**_CIBElement** - вспомогательный класс для работы с объектами, которые возвращает [CIBlockResult](../ciblockresult/index.md)::[GetNextElement](../ciblockresult/getnextelement.md). Для получения различных характеристик элемента рекомендуется использовать именно этот класс, т.к. использование методов этого объекта позволяет более гибко и эффективно работать с элементами информационных блоков.


#### Методы класса


| Метод | Описание | С версии |
| --- | --- | --- |
| [GetFields](getfields.md) | Возвращает массив [полей элемента](../../fields.md#felement), с замененными ссылками в полях *DETAIL_PAGE_URL* и *LIST_PAGE_URL*. | 3.1.3 |
| [GetProperties](getproperties.md) | Возвращает все или некоторые значения свойств элемента. | 3.1.3 |
| [GetProperty](getproperty.md) | Возвращает значения свойства элемента. | 3.1.3 |
| [GetGroups](getgroups.md) | Возвращает группы, к которым привязан элемент. | 3.1.3 |

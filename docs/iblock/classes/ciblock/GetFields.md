# GetFields


```
array
CIBlock::GetFields(
	int ID
);
```

Метод возвращает описание полей элементов инфоблоков. Структура массива описана в [CIBlock::SetFields](SetFields.md). Метод статический.


#### Параметры вызова


| Параметр | Описание |
| --- | --- |
| ID | Код информационного блока. |


#### Возвращаемое значение

Массив.


#### Смотрите также


- [CIBlock::SetFields](SetFields.md)

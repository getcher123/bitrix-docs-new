# SetMessages


```
CIBlock::SetMessages(
	int ID,
	array arFields
);
```

Метод устанавливает значения [дополнительных полей](../../fields.md#fiblocklang) инфоблока. Вызывается в методах [CIBlock::Add](add.md) и [CIBlock::Update](update.md). Нестатический метод.

**Примечание**: значения полей не указанных в параметре arFields сохраняются.


#### Параметры вызова


| Параметр | Описание |
| --- | --- |
| ID | Код инфоблока |
| arFields | Массив вида array("Поле" => "Значение" ...) |


#### Возвращаемое значение

Метод ничего не возвращает.

# IndexTemporaryTables


```
bool
CIBlockXMLFile::IndexTemporaryTables(
	bool with_sess_id = false
);
```

Индексация таблиц для ускорения доступа. Необходимо вызвать после загрузки данных из файла в таблицы БД, но до обработки этих данных. Нестатический метод.


#### Параметры вызова


| Параметр | Описание | С версии |
| --- | --- | --- |
| with_sess_id | Если значение *true*, то будут создаваться временные таблицы с поддержкой нескольких сессий "одновременного" импорта. Введён для совместимости. Необязательный параметр. | 8.5.5 |


#### Возвращаемое значение

Если во время создания индексов произойдет ошибка БД, метод вернет false.

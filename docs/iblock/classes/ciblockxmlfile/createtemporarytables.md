# CreateTemporaryTables


```
bool
CIBlockXMLFile::CreateTemporaryTables(
	bool with_sess_id = false
);
```

Создает таблицы для загрузки XML. Нестатический метод.

**Примечание:**для MySQL если определена константа MYSQL_TABLE_TYPE ( Специальные константы), то таблицы будут созданы заданного ей типа.


#### Параметры вызова


| Параметр | Описание | С версии |
| --- | --- | --- |
| with_sess_id | Если значение *true*, то будут создаваться временные таблицы с поддержкой нескольких сессий "одновременного" импорта. Введён для совместимости. Необязательный параметр. | 8.5.5 |


#### Возвращаемое значение

В случае, если таблицы создать не удалось, метод возвращает false.

# DropTemporaryTables


```
boolCIBlockXMLFile::DropTemporaryTables();
```

Удаляет таблицы, содержащие ранее загруженный файл. Необходимо вызывать метод перед началом загрузки XML. Нестатический метод.


#### Параметры вызова

Отсутствуют.
#### Возвращаемое значение

В случае возникновения ошибки метод возвращает false.

# UpdateSearch


```
CIBlockElement::UpdateSearch(
	int ID,
	bool bOverWrite = false
);
```

Индексирует элемент с кодом *ID*в модуле поиска. Нестатический метод.
#### Параметры вызова


| Параметр | Описание |
| --- | --- |
| ID | Код элемента. |
| bOverWrite | Переиндексировать ли элемент, если дата его изменения не изменилась. |

# CIBlockElement

**CIBlockElement** - класс для работы с элементами информационных блоков.


#### Методы класса


| Метод | Описание | С версии |
| --- | --- | --- |
| [GetList](getlist.md) | Возвращает список элементов по фильтру. | 3.0.8 |
| [GetByID](getbyid.md) | Возвращает элемент по его ID. | 3.0.8 |
| [GetElementGroups](getelementgroups.md) | Возвращает группы, которым принадлежит элемент, по его коду. | 3.1.3 |
| [GetIBlockByID](getiblockbyid.md) | Метод возвращает инфоблок по *ID* его элемента. | 3.1.3 |
| [GetProperty](getproperty.md) | Получение свойств злемента. | 3.0.8 |
| [Add](add.md) | Добавляет новый элемент. | 3.0.8 |
| [Update](update.md) | Изменяет элемент. | 3.0.8 |
| [Delete](delete.md) | Удаляет элемент из информационного блока. | 3.0.5 |
| [SetElementSection](setelementsection.md) | Привязывает элемент информационного блока к группам. | 3.1.3 |
| [SetPropertyValues](setpropertyvalues.md) | Сохраняет свойства элемента информационного блока. | 3.0.5 |
| [SetPropertyValuesEx](setpropertyvaluesex.md) | Сохраняет значения всех свойств элемента информационного блока. В отличие от [SetPropertyValues](setpropertyvalues.md) может не содержать полный набор значений. | 6.0.0 |

| [CounterInc](counterinc.md) | Увеличивает счетчик показов элемента. | 3.3.8 |

| [getPublicElementsOrmFilter](getpublicelementsormfilter.md) | * Метод для дополнения фильтра orm на основе массива. | 21.300.100 |

* - Методы работы с символьными кодами. Они работают, только если в настройках инфоблока включена опция **Транслитерировать из названия при добавлении элемента** для поля Символьный код элемента.

Параметры транслитерации берутся из настроек инфоблока, но могут быть переопределены в момент использования. Исключение - опция **Использовать внешний сервис для перевода**. В этом случае методы не работают, возвращают *null*.

Язык, с которого осуществляется транслитерация, выбирается из настроек сайта, к которому привязан инфоблок. Может быть переопределен в момент использования. Если сайты, к которым привязан инфоблок, имеют различные языки, то язык транслитерации ОБЯЗАТЕЛЬНО необходимо указать при вызове методов.

# Классы

Классы модуля **Информационные блоки**.


| Класс | Описание | С версии |
| --- | --- | --- |
| [CIBlock](ciblock/index.md) | Класс для работы с информационными блоками. | 3.0.3 |
| [CIBlockCMLImport](ciblockcmlimport/index.md) | Класс для импорта данных инфоблока в xml-формате. | 6.5.0 |
| [CIBlockElement](ciblockelement/index.md) | Класс для работы с элементами информационных блоков. | 3.0.5 |
| [CIBlockFormatProperties](ciblockformatproperties/dateformat.md) | Конвертирует дату в нужный формат. | 5.9.0 |
| [CIBlockPriceTools](ciblockpricetools/canbuy.md) |  | 5.9.0 |
| [CIBlockProperty](ciblockproperty/index.md) | класс для работы со свойствами информационных блоков. | 3.0.3 |
| [CIBlockPropertyEnum](ciblockpropertyenum/index.md) | Класс для работы с вариантами значений для свойств типа "список". | 3.1.3 |
| [CIBlockResult](ciblockresult/index.md) | Вспомогательный класс для работы с объектами результатов выборок. | 3.0.5 |
| [CIBlockRSS](ciblockrss/index.md) | Класс для работы с RSS лентами. | 3.0.14 |
| [CIBlockSection](ciblocksection/index.md) | Класс для работы с группами информационных блоков. | 3.0.4 |
| [CIBlockType](ciblocktype/index.md) | Класс для работы с типами информационных блоков. | 3.1.3 |
| [CIBlockXMLFile](ciblockxmlfile/index.md) | Класс для работы с файлами XML. | 6.5.0 |
| [Пользовательские свойства](user_properties/index.md) | Позволяют изменять представление (формы ввода и т.п.) стандартных свойств расширяя их возможности. | 5.1.0 |
| [_CIBElement](cibelement/index.md) | Вспомогательный класс для работы с объектом выборки элемента. | 3.1.3 |

**Примечание:**при использовании методов учтите, что время в полях используется в формате сайта, время в свойствах используется в формате: YYYY-MM-DD HH:MI:SS.

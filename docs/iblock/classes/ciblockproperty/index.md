# CIBlockProperty

**CIBlockProperty** - класс для работы со свойствами информационных разделов.


#### Методы класса


| Метод | Описание | С версии |
| --- | --- | --- |
| [GetList](getlist.md) | Возвращает список свойств по фильтру. | 3.0.3 |
| [GetByID](getbyid.md) | Возвращает свойство по его ID. | 3.0.3 |
| [Add](add.md) | Добавляет новое свойство. | 3.0.3 |
| [Update](update.md) | Изменяет параметры свойства. | 3.0.3 |
| [Delete](delete.md) | Удаляет свойства. | 3.0.3 |
| [GetPropertyArray](getpropertyarray.md) | Технический метод, возвращающий описание свойства с кодом *ID*. | 3.0.3 |
| [GetPropertyEnum](getpropertyenum.md) | Начинает перечисление свойств. | 3.1.3 |
| [GetUserType](GetUserType.md) | Метод возвращает описание пользовательского типа. | 5.1.0 |
| [UpdateEnum](updateenum.md) | Обновляет информацию в массиве результатов перечисления свойства. | 3.1.3 |

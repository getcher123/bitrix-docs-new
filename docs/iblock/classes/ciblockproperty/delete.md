# Delete


```
bool CIBlockProperty::Delete(
	int ID
);
```

Метод удаляет свойство и все его значения. Метод статический.


#### Параметры вызова


| Параметр | Описание |
| --- | --- |
| ID | Код свойства. |


#### Возвращаемое значение

В случае успешного удаления возвращается true, иначе - false.

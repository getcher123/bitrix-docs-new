# CIBlockPropertyEnum

**CIBlockPropertyEnum** - класс для работы со вариантами значений свойств типа "список".


#### Методы класса


| Метод | Описание | С версии |
| --- | --- | --- |
| [GetList](getlist.md) | Возвращает список вариантов свойств по фильтру. | 3.1.3 |
| [Add](add.md) | Добавляет новый вариант свойства. | 3.1.3 |
| [Update](update.md) | Изменяет вариант свойства. | 3.1.3 |
| [Delete](delete.md) | Удаляет вариант свойства. | 3.1.3 |
| [GetByID](GetByID.md) | Метод возвращает значения [всех полей](../../fields.md#fpropertyenum) варианта значения свойства типа "список". | 5.1.2 |

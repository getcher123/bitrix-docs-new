# CIBlockType

CIBlockType - класс для работы с типами информационных блоков.


#### Методы класса


| Метод | Описание | С версии |
| --- | --- | --- |
| [GetList](getlist.md) | Возвращает список типов по фильтру. | 3.1.3 |
| [GetByID](getbyid.md) | Возвращает тип по идентификатору. | 3.1.3 |
| [GetByIDLang](getbyidlang.md) | Возвращает настройки типа для заданного языка. | 3.1.3 |
| [Add](add.md) | Добавляет новый тип информационных блоков. | 3.1.3 |
| [Update](update.md) | Изменяет тип. | 3.1.3 |
| [Delete](delete.md) | Удаляет тип информационных блоков. | 3.1.3 |

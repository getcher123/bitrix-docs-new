# CIBlockRSS

**CIBlockRSS** - класс для работы с RSS лентами.


#### Методы класса


| Метод | Описание | С версии |
| --- | --- | --- |
| [GetNewsEx](getnewsex.md) | Загружает xml c указанного адреса и разбирает его в массив. | 3.1.2 |
| [FormatArray](formatarray.md) | Метод преобразует результат метода [GetNewsEx](getnewsex.md). | 3.0.15 |
| [XMLDate2Dec](xmldate2dec.md) | Преобразует дату из rss формата. | 3.0.16 |

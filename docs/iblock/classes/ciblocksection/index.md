# CIBlockSection

**CIBlockSection** - класс для работы с разделами (группами) информационных блоков.


#### Методы класса


| Метод | Описание | С версии |
| --- | --- | --- |
| [GetList](getlist.md) | Возвращает список разделов по фильтру. | 3.0.6 |
| [GetMixedList](getmixedlist.md) | Создаёт список разделов и элементов. | 5.1.0 |
| [GetByID](getbyid.md) | Возвращает параметры раздела по его коду ID. | 3.0.4 |
| [Add](add.md) | Добавляет в информационный блок новый раздел. | 3.0.6 |
| [Update](update.md) | Изменяет параметры существующего раздела. | 3.0.6 |
| [Delete](delete.md) | Удаляет раздел из информационного блока. | 3.0.4 |
| [GetCount](getcount.md) | Возвращает количество подразделов. | 3.0.6 |
| [GetSectionElementsCount](getsectionelementscount.md) | Возвращает количество элементов в разделе. | 3.2.1 |
| [GetTreeList](gettreelist.md) | Возвращает разделы отсортированные в порядке "полностью развернутого дерева" | 3.0.4 |
| [GetNavChain](getnavchain.md) | Возвращает путь от заданного раздела до корневого. | 3.0.4 |
| [ReSort](resort.md) | Пересчет левой и правой границ. | 3.0.4 |
| [createMnemonicCode](createmnemoniccode.md) | * Метод создания символьного кода. | 21.300.100 |
| [generateMnemonicCode](generatemnemoniccode.md) | * Метод генерации символьного кода. | 21.300.100 |
| [isExistsMnemonicCode](isexistsmnemoniccode.md) | * Метод проверки существования символьного кода. | 21.300.100 |

* - Методы работы с символьными кодами. Методы работают, только если в настройках инфоблока включена опция **Транслитерировать из названия при добавлении раздела** для поля Символьный код раздела.

Параметры транслитерации берутся из настроек инфоблока, но могут быть переопределены в момент использования. Исключение - опция **Использовать внешний сервис для перевода**. В этом случае методы не работают, возвращают *null*.

Язык, с которого осуществляется транслитерация, выбирается из настроек сайта, к которому привязан инфоблок. Может быть переопределен в момент использования. Если сайты, к которым привязан инфоблок, имеют различные языки, то язык транслитерации ОБЯЗАТЕЛЬНО необходимо указать при вызове методов.

# OnIBlockPropertyBuildList


```
arrayфункция-обработчик();
```

Событие вызывается при построении списка [пользовательских свойств](../classes/user_properties/index.md). Обработчик этого события должен вернуть массив описывающий пользовательское свойство. Содержимое этого массива описано в методе [GetUserTypeDescription](../classes/user_properties/GetUserTypeDescription.md).


#### Параметры

Отсутствуют.

#### Возвращаемое значение

Массив.

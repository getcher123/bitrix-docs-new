# Форум в ядре D7

Список классов модуля **Форум** в ядре D7.


| Класс, пространство имён | Описание |
| --- | --- |
| **UserTable** | Классы для работы с таблицей пользователей форума. Связь с таблицей пользователей - один к одному. |
| **MessageTable** | Классы для работы с таблицей сообщений. Связь с таблицей тем форумов - один к одному. |
| **Comments** | Классы для работы с форумными данными, как с комментариями. |
| **ForumTable** | Классы для работы с таблицей форумов. Связь с таблицей групп форумов - много к одному. |
| **GroupTable** | Классы для работы с таблицей групп форумов. |
| **ForumSiteTable** | Классы для работы с таблицей, связывающей форум с конкретным сайтом. Привязка форума к сайту по типу многие ко многим. |
| **Replica** | Пространство имён для классов репликации модуля Форум. |
